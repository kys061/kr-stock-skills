# 한국 역사적 버블 사례

## 1. 1999-2000 IT 버블

### 상황
- 닷컴 열풍이 한국에도 전파
- 코스닥 지수 1999년 1월 ~60 → 2000년 3월 ~289 (4.8배)
- 개인 투자자 대거 유입, 인터넷 관련주 투기

### 정량 지표 (당시 기준)
| 지표 | 수준 | 점수 |
|------|------|:----:|
| 시장 심리 | 극도의 낙관 | 2 |
| 레버리지 | 신용잔고 급증 | 2 |
| IPO 과열 | 코스닥 대거 상장 | 2 |
| 시장폭 이상 | IT주 쏠림, 시장폭 축소 | 2 |
| 가격 가속화 | 3개월 +80% (코스닥) | 2 |
| PER 밴드 | 코스닥 PER 100+ | 2 |
| **정량 합계** | | **12/12** |

### 결과
- 코스닥: 289 → 52 (-82%, 2001년)
- 코스피: 1,059 → 468 (-56%)

| 가격 가속화 | 3개월 +15% | 1 |
| PER 밴드 | KOSPI PER ~14 | 2 |
| **정량 합계** | | **6/12** |

### 정성 조정
| 조정 | 근거 | 점수 |
|------|------|:----:|

## 3. 2021년 동학개미 버블

### 상황
- 코로나 이후 글로벌 유동성 폭발
- 한국 개인 투자자 대거 유입 ("동학개미")
- KOSPI 2021년 1월 사상 최초 3,000 돌파 → 6월 3,316 고점

### 정량 지표
| 지표 | 수준 | 점수 |
|------|------|:----:|
| 시장 심리 | VKOSPI ~17 (낮은 편) | 1 |
| 레버리지 | 신용잔고 사상 최고 (~25조) YoY +40% | 2 |
| IPO 과열 | 카카오뱅크, SK바이오팜 등 대형 IPO | 2 |
| 시장폭 이상 | 일부 다이버전스 | 1 |
| 가격 가속화 | 3개월 +20% | 2 |
| PER 밴드 | KOSPI PER ~13-14 | 1 |
| **정량 합계** | | **9/12** |

### 정성 조정
| 조정 | 근거 | 점수 |
|------|------|:----:|
| 사회적 침투 | 주식 투자 앱 다운로드 급증, 20대 투자자 급증 | +1 |
| 미디어 트렌드 | "주린이" 신조어, 지상파 투자 프로그램 | +1 |
| 밸류에이션 괴리 | 적자 기업 급등, 테마주 투기 | +1 |
| **정성 합계** | | **+3** |

**최종 점수**: 12/15 → Euphoria (유포리아)

### 결과
- KOSPI: 3,316 → 2,155 (-35%, 2022년 9월)
- 신용잔고 반대매매 급증이 하락 가속
- 외국인 2021년 하반기부터 대규모 순매도

## 4. 패턴 요약

### 한국 버블의 공통 특징
1. **개인 투자자 급증** — 항상 동반
2. **신용잔고 급증** — 2/3 사례에서 사상 최고
3. **외국인 이탈 선행** — 고점 3-6개월 전 시작
4. **"이번엔 다르다" 내러티브** — 항상 존재
5. **PER 상단** — KOSPI PER 13+ 구간에서 주의

### 한국 버블의 고유 특성 (US와의 차이)
- **반대매매 연쇄**: 한국 특유의 하락 가속 메커니즘
- **외국인 자금 흐름**: 미국보다 예측 가능 (일별 데이터 공개)
- **Korea Discount**: 절대 PER 수준이 미국보다 낮아도 버블 가능
- **정책 개입**: 공매도 금지, 증거금률 인상 등 정부 개입이 잦음
