# kr-bubble-detector (한국 시장 버블 탐지기)

## 개요

| 항목 | 값 |
|------|-----|
| US 원본 | us-market-bubble-detector (v2.1) |
| 복잡도 | Medium |
| 시간 지평 | 수개월~수년 (구조적/장기) |
| 탐지 대상 | 30%+ 폭락 가능성이 있는 버블 과열 |
| 방법론 | Minsky/Kindleberger 프레임워크 v2.1 |
| 특징 | 정량(max 12점) + 정성(max 3점) 2단계 평가 |

## 방법론

### 2단계 평가 프로세스

**Phase 2 (정량)**: 6개 객관적 지표로 0-12점 자동 스코어링
**Phase 3 (정성)**: 3개 주관적 조정으로 최대 +3점 추가

정량 데이터를 먼저 수집하고, 그 위에 정성 판단을 겹치는 구조.
확증 편향을 방지하기 위해 정성 조정은 엄격한 기준으로 제한.

## 6개 정량 지표

각 지표는 0점, 1점, 2점 중 하나. 합계 최대 12점.

### 지표 1: VKOSPI + 시장 위치 (자만+과열 복합)

| 조건 | 점수 |
|------|:----:|
| VKOSPI < 13 AND KOSPI 52주 고점 대비 5% 이내 | **2** |
| VKOSPI 13-18 AND KOSPI 고점 근접 | **1** |
| VKOSPI > 18 OR 고점 대비 10%+ 하락 | **0** |

### 지표 2: 신용잔고 변화 (레버리지)

| 조건 | 점수 |
|------|:----:|
| 신용잔고 YoY +20% 이상 AND 역대 최고 | **2** |
| 신용잔고 YoY +10~20% | **1** |
| YoY +10% 이하 또는 감소 | **0** |

### 지표 3: IPO 과열도

| 조건 | 점수 |
|------|:----:|
| 분기 IPO 건수 > 5년 평균 2배 AND 평균 청약경쟁률 > 500:1 | **2** |
| IPO 건수 > 5년 평균 1.5배 | **1** |
| 정상 수준 | **0** |

### 지표 4: 시장폭 이상 (Breadth Anomaly)

| 조건 | 점수 |
|------|:----:|
| KOSPI 신고가 AND 200MA 위 종목 비율 < 45% | **2** |
| 200MA 위 종목 비율 45-60% | **1** |
| 200MA 위 종목 비율 > 60% | **0** |

### 지표 5: 가격 가속화 (Price Acceleration)

| 조건 | 점수 |
|------|:----:|
| KOSPI 3개월 수익률 > 10년 95th percentile | **2** |
| 85-95th percentile | **1** |
| 85th percentile 미만 | **0** |

### 지표 6: KOSPI PER 밴드 (한국 특화)

| 조건 | 점수 |
|------|:----:|
| KOSPI PER ≥ 14 (역사적 상위 10%) | **2** |
| KOSPI PER 12-14 (상위 10-25%) | **1** |
| KOSPI PER < 12 (중위 이하) | **0** |

KOSPI 역사적 PER 범위: 저점 ~8 (2008, 2020), 중위 ~10-11, 고점 ~14-16 (2007, 2021)

## 3개 정성 조정

각 조정은 +0 또는 +1점. 합계 최대 +3점.
**엄격한 기준**: 모든 하위 요건을 충족해야 +1점.

### 조정 A: 사회적 침투도 (0-1점)

+1점 요건 (3가지 **모두** 필요):
- 직접 사용자 보고 (지인의 주식 투자 언급)
- 구체적 사례 (택시기사, 학생 등 비전문가 투자)
- 복수 출처 (3개 이상 독립 소스)

### 조정 B: 미디어/검색 트렌드 (0-1점)

+1점 요건 (2가지 **모두** 필요):
- 네이버 트렌드 "주식투자" 검색량 YoY 5배+
- 지상파 방송 (KBS/MBC/SBS) 투자 관련 보도 확인

### 조정 C: 밸류에이션 괴리 (0-1점)

+1점 요건 (3가지 **모두** 필요):
- KOSPI PER > 14 (or KOSDAQ PER > 30)
- 실적 무시 상승 (적자 기업 급등)
- "이번엔 다르다" 내러티브 확인

## 리스크 존 매핑

| 점수 | 존 | 리스크 예산 | 행동 지침 |
|:----:|-----|:---------:|----------|
| 0-4 | Normal (정상) | 100% | 정상 투자 전략 |
| 5-7 | Caution (주의) | 70-80% | 부분 이익실현 시작 |
| 8-9 | Elevated Risk (위험 상승) | 50-70% | 이익실현 강화, 선별 진입 |
| 10-12 | Euphoria (유포리아) | 40-50% | 적극 이익실현 |
| 13-15 | Critical (위기) | 20-30% | 최대 방어, 헤지 |

## 실행 방법

```bash
cd ~/stock/skills/kr-bubble-detector/scripts
python kr_bubble_detector.py --output-dir ./output
```

### 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| `--output-dir` | 리포트 출력 디렉토리 | `./output` |
| `--breadth-json` | kr-market-breadth JSON (선택) | 자체 계산 |
| `--qualitative` | 정성 조정 JSON 문자열 (선택) | 모두 0점 |

## 출력 파일

- `kr_bubble_YYYY-MM-DD_HHMMSS.json` — 구조화된 분석 데이터
- `kr_bubble_YYYY-MM-DD_HHMMSS.md` — 사람이 읽을 수 있는 리포트
