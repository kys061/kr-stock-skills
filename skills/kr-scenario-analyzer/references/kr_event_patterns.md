# 한국 이벤트 패턴

## BOK 금통위

- **주기**: 연 8회 (1.5개월 간격)
- **패턴**: 금리 결정 전 1주 변동성 확대, 결정 후 방향성
- **주요 영향 섹터**: 금융, 건설/부동산, 유통/소비재

## 북한 리스크

- **패턴**: 갑작스러운 KOSPI 하락 → 1-3일 내 회복 (과거 패턴)
- **방산주**: 이벤트 당일 급등
- **외국인**: 단기 순매도 후 복귀

## 반도체 사이클

- **주기**: 3-5년
- **리드 지표**: 메모리 현물가, DRAM 가격, 삼성전자 실적
- **전파**: 삼성전자 → 장비/소재주 → KOSPI 전체

## 중국 통상 리스크

- **패턴**: 한중 관계 악화 시 화학/철강 타격
- **대안**: IT/소프트웨어, 방산 수혜
- **회복**: 정치적 해결 시 빠른 반등
