# 한국 VCP 패턴 가이드

## VCP (Volatility Contraction Pattern)
Mark Minervini의 대표적 매수 패턴.
변동성이 수축하며 거래량이 감소하면 브레이크아웃 임박 신호.

## 한국 적응 포인트

### ±30% 가격제한폭 영향
- 한국은 일일 ±30% 가격제한 → 변동성이 구조적으로 제한
- T1(첫 번째 수축) 깊이: **10-30%** (US 8-35%)
- 소형주 최대 깊이: **40%** (US 50%)

### RS 벤치마크
- US: S&P 500 대비
- KR: **KOSPI** 대비 (KOSDAQ 종목도 KOSPI 대비 계산)

## 패턴 식별 기준

### 수축 (Contraction) 정의
1. 스윙 고점에서 하락 → 스윙 저점까지의 조정
2. 각 수축은 이전보다 얕아야 함 (수축 비율 ≤ 0.75)
3. 거래량은 수축 기간 동안 감소 (Dry-Up)

### 피봇 포인트
- 마지막 수축의 고가 = 피봇 포인트
- 진입: 피봇 돌파 + 거래량 확인
- 스탑: 마지막 수축의 저가 -1~2%
