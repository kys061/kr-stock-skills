# 한국 프로그램 매매 가이드

## 1. 프로그램 매매 구조

### 1.1 차익거래 (Arbitrage)
KOSPI200 선물과 현물 간 가격 괴리를 이용한 매매.
- **매수 차익**: 선물 매도 + 현물(바스켓) 매수 → 콘탱고 해소
- **매도 차익**: 선물 매수 + 현물 매도 → 백워데이션 해소

### 1.2 비차익거래 (Non-Arbitrage)
차익과 무관한 바스켓 매매 (기관 자산배분, ETF 설정/환매 등).
비차익 연속 매도가 5일 이상이면 시장 하방 압력 경고.

### 1.3 금액 기준
| 유형 | 유의미 | 대규모 |
|:----:|:------:|:------:|
| 차익 | 5,000억 | 1조 |
| 비차익 | 3,000억 | 5,000억 |

## 2. 선물 베이시스

### 2.1 정의
베이시스 = 선물가격 - 현물가격 (또는 %)

### 2.2 이론 베이시스
F = S × (1 + r × t/365)
- S: 현물 지수
- r: 무위험이자율 (3.5%)
- t: 잔존 일수

### 2.3 상태 분류
| 상태 | 베이시스 % | 의미 |
|:----:|:---------:|------|
| DEEP_CONTANGO | +1.5%+ | 강한 콘탱고 → 차익매도 가능 |
| CONTANGO | +0.7~1.5% | 선물 프리미엄 |
| FAIR | ±0.3% | 이론가 근접 |
| BACKWARDATION | -0.7~-0.3% | 선물 할인 |
| DEEP_BACKWARDATION | -1.5%- | 강한 백워데이션 → 차익매수 가능 |

## 3. 만기일 효과

### 3.1 만기 유형
- **월간 만기**: 매월 둘째 주 목요일 (옵션)
- **분기 만기**: 3/6/9/12월 둘째 주 목요일 (선물+옵션, 쿼드러플 위칭)

### 3.2 주요 패턴
- Pin Risk: 최대고통가격 근접 시 가격 고정 경향
- Gamma Squeeze: 옵션 감마 급증으로 변동성 확대
- Rollover Pressure: 선물 롤오버 매매 압력
- Expiry Volatility: 만기일 거래량 급증 + 변동성
