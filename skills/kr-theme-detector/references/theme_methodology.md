# 테마 탐지 방법론

## 1. 3D 스코어링 모델

### 1.1 Dimension 1: Theme Heat (0-100)

테마의 현재 모멘텀/열기를 0-100으로 정량화.

#### Components
| 컴포넌트 | 가중치 | 계산 방법 |
|----------|:------:|----------|
| Momentum | 40% | 대표종목 시총가중 1주/1개월 수익률 |
| Volume | 20% | 최근 5일 거래량 / 20일 평균 거래량 |
| Uptrend Ratio | 25% | 200MA 위 대표종목 비율 |
| Breadth | 15% | 최근 5일 양봉 비율 |

#### Momentum 점수 변환
| 가중 수익률 (1주) | 점수 |
|:-----------------:|:----:|
| ≥ 5% | 95 |
| ≥ 3% | 80 |
| ≥ 1% | 65 |
| ≥ 0% | 50 |
| ≥ -1% | 35 |
| ≥ -3% | 20 |
| < -3% | 5 |

### 1.2 Dimension 2: Lifecycle Maturity

| 단계 | 조건 |
|------|------|
| Early | 모멘텀 시작 (1주 > 0, 1개월 ≤ 5%), 거래량 증가 초기 (ratio < 1.5) |
| Mid | 모멘텀 지속 (1개월 > 5%), 거래량 안정적 (1.0~2.0), 업트렌드 50%+ |
| Late | 모멘텀 극단 (1개월 > 15%), 거래량 급증 (ratio > 2.0), 업트렌드 80%+ |
| Exhaustion | 1주 수익률 하락 but 1개월 여전히 높음, 거래량 감소 |

### 1.3 Dimension 3: Confidence

| 수준 | 조건 |
|------|------|
| High | 업트렌드 > 60% AND 거래량 확인 (ratio > 1.2) |
| Medium | 업트렌드 40~60% OR 거래량 미확인 |
| Low | 업트렌드 < 40% AND 거래량 미확인 |

## 2. 방향 판정 (Direction)

```
Bullish: 시총가중수익률 > 0 AND (업트렌드 > 50% OR 거래량비 > 1.3)
Bearish: 시총가중수익률 < 0 AND (업트렌드 < 50% OR 거래량비 < 0.8)
Neutral: 위 조건에 해당하지 않음
```

## 3. 한국 시장 특화

### 3.1 테마 정의 (kr_themes.yaml)
14개 핵심 테마를 대표종목 + 업종 기반으로 정의.
각 테마의 core/sub 종목 구분.

### 3.2 한국 특화 라이프사이클 징후
- 개인투자자 순매수 급증 = Late/Exhaustion 징후
- 관련 ETF 신규 상장 = Mid→Late 전환 징후
- 뉴스 빈도 급증 = Late 징후

### 3.3 밸류에이션 참조
PER/PBR 데이터를 라이프사이클 판단 보조 지표로 활용.
과열 (PER > 업종평균 × 1.5) = Late 징후.
