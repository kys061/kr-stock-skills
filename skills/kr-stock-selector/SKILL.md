# kr-stock-selector: 주식종목선별 자동화

> 사용자 수동 관리 5가지 조건을 자동화하여 KOSPI/KOSDAQ 전 종목에서 조건 충족 종목을 선별한다.
> Pass/Fail 방식으로 통과 종목 리스트 + Watch List(4/5 통과)를 생성.

## 사용 시점

- 매일/매주 트렌드 조건을 충족하는 종목 리스트 확인
- Stage 2 상승 추세 유니버스 자동 관리
- 수동 종목선별 작업 대체

## 데이터 소스 우선순위

```
1순위: KRX Open API (시가총액, 종목 리스트) — Tier 0
2순위: yfinance (일봉 OHLCV 300일+, 배치 다운로드) — Tier 1
3순위: pykrx (yfinance 실패 시 OHLCV 폴백) — Tier 2
```

### 폴백 전략

| 단계 | 기본 소스 | 폴백 1 | 폴백 2 |
|------|----------|--------|--------|
| 유니버스 구축 | KRX Open API | yfinance (ETF 구성종목) | pykrx (종목 리스트) |
| OHLCV 수집 | yfinance 배치 | yfinance 개별 | pykrx |

## 5가지 선별 조건

| # | 조건명 | 상세 | 판정 |
|---|--------|------|------|
| 1 | 주가이평추세 | 200일 SMA 상승+보합 20일 연속 유지 | Pass/Fail |
| 2 | 주가이평배열 | 종가 > 50SMA > 150SMA > 200SMA (4중 정배열) | Pass/Fail |
| 3 | 52주 최저가대비 | 현재가가 52주 저가 대비 +30% 이상 | Pass/Fail |
| 4 | 52주 최고가대비 | 현재가가 52주 고가 대비 -25% 이내 | Pass/Fail |
| 5 | 시가총액 | 현재가 기준 1,000억원 이상 | Pass/Fail |

## 실행 절차

### Step 1: 환경 확인

```bash
# KRX API 키 확인 (없으면 yfinance 폴백으로 자동 진행)
echo $KRX_API_KEY
```

### Step 2: Python 스크립트 실행

```bash
cd ~/stock/skills/kr-stock-selector/scripts
python kr_stock_selector.py
```

옵션:
```bash
# KOSPI만
python kr_stock_selector.py --market KOSPI

# KOSDAQ만
python kr_stock_selector.py --market KOSDAQ

# 특정 날짜
python kr_stock_selector.py --date 2026-03-11
```

### Step 3: 결과 확인

- 성공 시 `reports/kr-stock-selector_market_주식종목선별_{YYYYMMDD}.md` 생성
- 실패 시 에러 메시지 출력

### Step 4: 이메일 발송

```bash
cd ~/stock && python3 skills/_kr_common/utils/email_sender.py "reports/kr-stock-selector_market_주식종목선별_{YYYYMMDD}.md" "kr-stock-selector"
```

## 리포트 구성

| 섹션 | 내용 |
|------|------|
| 요약 | 분석 대상 N개, 통과 종목 M개 (X%) |
| 통과 종목 | 종목코드/명/시장/시총/현재가/52주대비 (시총 내림차순) |
| 조건별 통과율 | 5개 조건별 통과 수와 비율 |
| Watch List | 4/5 통과 종목 + 미충족 조건/현재수치/필요수치 |

## 관련 스킬

| 스킬 | 차이점 |
|------|--------|
| kr-vcp-screener | Stage 2 + VCP 패턴 + RS Rank (더 복잡) |

## Output Rule (마크다운 리포트 저장)

- 분석 완료 후 결과를 **마크다운 파일로 저장**한다
- 저장 경로: `reports/kr-stock-selector_market_주식종목선별_{YYYYMMDD}.md`
- 파일 구조:
  ```markdown
  # 주식종목선별 리포트
  > 생성일: {YYYY-MM-DD HH:MM} | 대상: {시장}
  > 데이터 소스: KRX Open API (Tier 0) + yfinance (Tier 1) | 분석 유형: SCREENING

  ## 요약
  ## 통과 종목
  ## 조건별 통과율
  ## Watch List (4/5 통과)

  ---
  *Generated by kr-stock-selector*
  ```
- `reports/` 디렉토리가 없으면 자동 생성
- 동일 파일명이 존재하면 덮어쓰기 (같은 날 재분석 시)
