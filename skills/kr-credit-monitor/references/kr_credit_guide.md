# 한국 신용거래 분석 가이드

## 1. 신용거래 구조

### 1.1 신용융자
증권사에서 자금을 빌려 주식을 매수하는 거래.
- 최초 담보비율: 200% (자기자금 50% + 대출 50%)
- 담보유지비율: 140% (시가 하락 시 기준)

### 1.2 대주 (신용대주)
주식을 빌려 매도하는 거래 (공매도의 일종).

## 2. 반대매매 (강제청산)

### 2.1 발동 조건
담보비율이 140% 이하로 하락 시:
1. 증권사가 담보부족 통보
2. D+2 이내에 추가 담보 납입 또는 반대매매 집행
3. 시장가로 강제 매도 (당일 시가에 매도 주문)

### 2.2 시나리오별 영향
| 시장 하락 | 담보비율 변화 | 반대매매 규모 |
|:---------:|:-----------:|:-----------:|
| -10% | 200% → 178% | 소규모 |
| -20% | 200% → 156% | 보통 |
| -30% | 200% → 133% | 대규모 (140% 미달) |

### 2.3 연쇄 하락 메커니즘
반대매매 → 추가 하락 → 추가 반대매매 → ... (악순환)
시장 거래대금 대비 10% 이상이면 연쇄 하락 위험.

## 3. 레버리지 사이클

### 3.1 4단계
| 단계 | 조건 | 리스크 |
|:----:|------|:------:|
| EXPANSION | MoM↑ + YoY↑ | 증가 |
| PEAK | 퍼센타일 90%+ | 높음 |
| CONTRACTION | MoM↓ + YoY↑ | 감소 |
| TROUGH | 퍼센타일 20%- | 낮음 |

### 3.2 예탁금 대비 신용잔고 비율
예탁금(투자자 대기자금) 대비 신용잔고(레버리지) 비율.
- 80% 이상: 과열 (대기자금 대비 레버리지 과다)
- 60% 이상: 높음
- 40% 이하: 건전

## 4. 데이터 소스

- 금융투자협회: 신용잔고 일별 현황
- 한국거래소: 시가총액
- 한국예탁결제원: 투자자예탁금
