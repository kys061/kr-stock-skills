# 한국 시장 패턴 매트릭스

## 4가지 시장 패턴

### 1. POLICY_PIVOT (정책 전환 예측)
- **트리거**: 전환기 레짐 + 높은 전환 확률
- **원칙**: "중앙은행과 유동성에 집중하라"
- **주식 비중**: 70-90%
- **한국 적용**: BOK 금리 인하 시그널, 정부 부양책

### 2. UNSUSTAINABLE_DISTORTION (지속불가 왜곡)
- **트리거**: 위험 점수 ≥ 60 + 수축/인플레이션 레짐
- **원칙**: "틀렸을 때 얼마나 잃느냐가 가장 중요하다"
- **주식 비중**: 30-50%
- **한국 적용**: 외국인 대규모 이탈, 환율 급등

### 3. EXTREME_CONTRARIAN (극단 역발상)
- **트리거**: FTD 확인 + 높은 위험 + 약세 시장
- **원칙**: "가장 큰 수익은 약세장에서 나온다"
- **주식 비중**: 25-40%
- **한국 적용**: KOSPI 급락 후 FTD, 외국인 순매수 전환

### 4. WAIT_OBSERVE (관망)
- **트리거**: 낮은 확신 + 혼합 시그널
- **원칙**: "보이지 않으면 스윙하지 마라"
- **주식 비중**: 0-20%
- **한국 적용**: 미중 갈등 불확실, BOK 방향 불명확
