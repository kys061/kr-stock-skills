# 한국 시장 주간 전략 워크플로우 가이드

## 1. 주간 전략 프로세스

### 매주 월요일 개장 전
1. 지난주 성과 복기
2. 시장 환경 분류 (RISK_ON / BASE / CAUTION / STRESS)
3. 8개 체크리스트 확인
4. 시나리오 수립 (Base 50% / Bull 25% / Bear 25%)
5. 섹터 비중 조정 (±15% 제한)
6. 리스크 관리 점검
7. 주간 액션 플랜 확정

## 2. 시장 환경 판단 기준

### RISK_ON (강세장)
- KOSPI 상승 추세 (20일선 위)
- 외국인 순매수 지속
- 원화 강세 (USD/KRW 하락)
- 주요 경기지표 개선

### BASE (횡보장)
- KOSPI 횡보 (20일선 근처)
- 수급 혼합
- 환율 안정

### CAUTION (주의)
- KOSPI 하락 전환 가능
- 외국인 순매도 시작
- 환율 불안정

### STRESS (약세장)
- KOSPI 하락 추세 (60일선 아래)
- 외국인 대량 순매도
- 원화 급락

## 3. 겸업 투자자 운용 가이드

### 주 2-3시간 가용 기준
- 월요일: 환경 분석 + 주문 입력 (30분)
- 수요일: 중간 점검 (15분)
- 금요일: 주간 복기 (30분)

### 핵심 원칙
- 과도한 매매 금지 (주 2-3회 이내)
- 설정 가격 도달 시만 매매
- 감정적 매매 방지 (계획 외 매매 금지)
