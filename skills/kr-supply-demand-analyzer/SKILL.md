# kr-supply-demand-analyzer

한국 주식 시장의 시장/섹터 레벨 수급 동향과 유동성을 종합 분석하는 스킬.

## 개요

PyKRX 기반 시장 전체 투자자 매매동향과 섹터별 자금 흐름을 분석한다.
기존 kr-institutional-flow(종목 레벨 4팩터)와 달리, 시장/섹터 레벨의 거시적 수급 분석에 특화되었다.

## 핵심 기능

1. **시장 레벨 수급 분석** — KOSPI/KOSDAQ 전체 외국인/기관/개인 순매수 + 7단계 시그널
2. **섹터별 자금 흐름** — 14 한국 섹터 로테이션 + HHI 집중도 + 외국인-기관 괴리
3. **유동성 지표** — 거래대금 이동평균, 회전율, 상위종목 집중도
4. **수급 종합 스코어** — 4 컴포넌트 종합 (0-100)

## 스코어링

### 가중치
| 팩터 | 가중치 | 설명 |
|------|:------:|------|
| Market Flow | 30% | 시장 순매수 강도 |
| Sector Rotation | 25% | 섹터 로테이션 건전성 |
| Liquidity | 25% | 유동성 충분도 |
| Investor Sentiment | 20% | 투자자 심리 |

### 등급
| 등급 | 점수 | 의미 |
|------|:----:|------|
| STRONG_INFLOW | 80-100 | 강력 자금 유입 |
| INFLOW | 65-79 | 자금 유입 |
| BALANCED | 45-64 | 균형 |
| OUTFLOW | 30-44 | 자금 유출 |
| STRONG_OUTFLOW | 0-29 | 강력 자금 유출 |

### 시장 수급 시그널 (7단계)
| 시그널 | 최소 점수 | 의미 |
|--------|:--------:|------|
| STRONG_BUY | 85 | 강력 유입 |
| BUY | 70 | 유입 |
| MILD_BUY | 55 | 약한 유입 |
| NEUTRAL | 45 | 중립 |
| MILD_SELL | 30 | 약한 유출 |
| SELL | 15 | 유출 |
| STRONG_SELL | 0 | 강력 유출 |

## kr-institutional-flow와의 차이점

| 항목 | kr-institutional-flow | kr-supply-demand-analyzer |
|------|----------------------|--------------------------|
| 분석 대상 | 개별 종목 | 시장/섹터 전체 |
| 팩터 수 | 4 (외국인/기관/일관성/거래량) | 4 (시장수급/섹터/유동성/심리) |
| 시그널 | 5등급 | 7등급 (시장) + 5등급 (종합) |
| 유동성 | 없음 | 거래대금/회전율/집중도 |
| 섹터 분석 | 없음 | 14 섹터 HHI + 로테이션 |
