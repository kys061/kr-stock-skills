# 한국 공매도 분석 가이드

## 1. 공매도 기본 개념

공매도(Short Selling)는 주식을 빌려 매도한 후 나중에 되갚는 거래.
한국은 KRX에서 일별 공매도 잔고와 거래를 공시한다.

### 1.1 주요 지표
- **잔고비율**: 공매도 잔고 / 상장주식수 (%)
- **거래비율**: 공매도 거래량 / 총 거래량 (%)
- **Days-to-Cover (DTC)**: 공매도 잔고 / 평균 거래량 (일)

### 1.2 잔고비율 수준
| 비율 | 수준 | 의미 |
|:----:|:----:|------|
| 10%+ | 극단적 | 역사적 고수준, 숏스퀴즈 가능 |
| 5-10% | 높음 | 공매도 과열 주의 |
| 2-5% | 보통 | 일반적 수준 |
| 1-2% | 낮음 | 공매도 관심 낮음 |
| <1% | 미미 | 공매도 거의 없음 |

## 2. 숏커버 시그널

### 2.1 연속 감소 패턴
| 연속 감소일 | 강도 | 의미 |
|:----------:|:----:|------|
| 7일+ | 강한 | 본격적 숏커버 진행 |
| 5-6일 | 보통 | 숏커버 시작 |
| 3-4일 | 약한 | 일시적 감소 가능 |

### 2.2 급감
전일 대비 -10% 이상 잔고 감소 → 대규모 숏커버 발생

### 2.3 숏스퀴즈 조건
- 잔고비율 5% 이상
- 잔고 감소 추세
- 주가 상승 중
- DTC 5일 이상

## 3. 데이터 소스

- PyKRX: `stock.get_shorting_balance_by_date()` (일별 공매도 잔고)
- PyKRX: `stock.get_shorting_volume_by_date()` (일별 공매도 거래량)
