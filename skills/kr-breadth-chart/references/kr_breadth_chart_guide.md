# 한국 시장폭 차트 해석 가이드

## 1. 시장폭(Market Breadth)이란?

시장폭은 시장 전체의 참여 정도를 측정하는 지표.
지수가 상승해도 소수 종목만 오르면 "좁은 시장(narrow market)"이며,
다수 종목이 함께 오르면 "넓은 시장(broad market)"이다.

한국 시장에서는 삼성전자/SK하이닉스 등 대형주의 지수 영향력이 크기 때문에
시장폭 분석이 특히 중요하다.

## 2. 핵심 지표

### 2.1 200MA 위 종목 비율

가장 기본적인 시장폭 지표. KOSPI/KOSDAQ 전 종목 중
200일 이동평균선 위에 있는 종목의 비율.

**역사적 범위 (KOSPI)**:
- 극단적 강세: 75-85% (2009년 3월 반등, 2020년 11월)
- 정상 강세: 55-70%
- 중립: 40-55%
- 약세: 25-40%
- 극단적 약세: 10-25% (2008년 10월, 2020년 3월)

### 2.2 50MA 위 종목 비율

단기 시장폭. 200MA보다 민감하게 반응.
FTD(Follow-Through Day) 전후에 이 지표의 개선이 중요.

### 2.3 신고가-신저가 차이 (NH-NL)

52주 신고가 종목 수 - 52주 신저가 종목 수.
양(+)이면 건강, 음(-)이면 악화.

## 3. 차트 패턴 가이드

### 3.1 건전한 상승장 패턴
```
KOSPI   ─────────/─────────/──────
Breadth ────────/──────────/──────
           (동행 상승 = 건전)
```
- 지수와 시장폭이 함께 상승
- 200MA 위 종목 비율 50%+ 유지
- 업종별 고르게 상승

### 3.2 천장 형성 패턴
```
KOSPI   ──────────/\──/\──────────
Breadth ──────/\──────/───────────
           (다이버전스 = 위험)
```
- 지수 신고가이지만 시장폭은 하락
- Phase 2 → Phase 3: kr-market-top-detector의 Component 4
- 외국인 순매도 동반 시 위험 증폭

### 3.3 바닥 형성 패턴
```
KOSPI   ──────────\/──\/──────────
Breadth ──────────\/────/─────────
           (시장폭 선행 개선 = 바닥)
```
- 지수 저점이지만 시장폭이 먼저 개선
- Phase 2 → Phase 3: kr-ftd-detector의 Component 4
- 외국인 순매수 전환 시 시그널 강화

### 3.4 V자 반등 패턴
```
KOSPI   ──────\        /──────────
Breadth ────\  \      / /─────────
              \  \  /  /
               \/  \/
           (급락 후 동반 급반등)
```
- 20-30% 급락 후 V자 반등
- 시장폭이 20% 이하에서 급반등 시 강한 매수 시그널
- 역사적 사례: 2020년 3월, 2008년 10월 이후

## 4. 한국 시장 특수 고려사항

### 4.1 삼성전자 효과
- KOSPI 시가총액의 ~20-25%를 차지
- 삼성전자 단독 움직임으로 지수가 움직여도 시장폭은 변하지 않음
- 시장폭 분석이 "진짜 시장 상태"를 보여주는 이유

### 4.2 KOSPI vs KOSDAQ 괴리
- KOSPI: 대형 우량주 중심, 외국인 비중 높음
- KOSDAQ: 중소형 성장주, 개인 비중 높음
- 두 시장의 시장폭이 괴리될 때 → 시장 분열 시그널

| 패턴 | 해석 |
|------|------|
| KOSPI 강 + KOSDAQ 약 | 대형주 쏠림, 소형주 소외 |
| KOSPI 약 + KOSDAQ 강 | 개인 투기, 테마주 장세 |
| 양쪽 모두 강 | 건전한 강세장 |
| 양쪽 모두 약 | 약세장 / 조정 |

### 4.3 외국인 매매와 시장폭

한국 시장에서 외국인 매매동향은 시장폭에 직접적 영향:
- 외국인 연속 순매수 → 대형주 시장폭 개선
- 외국인 연속 순매도 → 대형주 시장폭 악화
- 기관 순매수 + 외국인 순매수 동시 → 가장 건전한 시장폭

### 4.4 옵션 만기일 효과 (매월 둘째 목요일)

옵션 만기일 전후로 시장폭이 일시적으로 왜곡될 수 있음:
- 만기 전: 프로그램 매매로 인한 비정상적 등락
- 만기 후: 정상화 (진짜 시장폭 확인 가능)
- 분석 시 만기일 ±2일 데이터는 주의해서 해석

## 5. 실전 체크리스트

차트 분석 시 순서대로 확인:

- [ ] **Step 1**: KOSPI Breadth (200MA 위 비율) 현재 수준은?
  - Strong (70%+) / Normal (50-70%) / Weak (30-50%) / Bear (<30%)
- [ ] **Step 2**: 추세 방향은?
  - 상승 / 하락 / 횡보
- [ ] **Step 3**: 지수와의 다이버전스가 있는가?
  - 약세 다이버전스 (지수↑ + Breadth↓) → 위험
  - 강세 다이버전스 (지수↓ + Breadth↑) → 기회
- [ ] **Step 4**: KOSDAQ Breadth와 괴리가 있는가?
  - 괴리 → 시장 분열, 주의
- [ ] **Step 5**: 업종별 히트맵에서 로테이션 패턴은?
  - 방어→성장: 리스크 온 전환
  - 성장→방어: 리스크 오프 전환
- [ ] **Step 6**: 외국인 매매동향과 일치하는가?
  - 외국인 순매수 + Breadth 개선 = 건전
  - 외국인 순매도 + Breadth 악화 = 위험
