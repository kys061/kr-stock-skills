# 한국 배당주 투자 5-Step SOP 가이드

## 1. Step 1: SCREEN (스크리닝)

### 필수 조건
- 배당수익률 ≥ 2.5%
- 최소 3년 연속 배당
- 최근 3년 감배 없음
- 배당성향 ≤ 80%
- 시가총액 ≥ 5,000억원
- 부채비율 ≤ 150%
- 유동비율 ≥ 1.0
- ROE ≥ 5%
- 매출/EPS 3년 양의 추세

### 한국 배당주 특성
- 연 1회 배당 (12월 결산 기업 대부분)
- 평균 배당수익률 2~3% (US 1.5~2% 대비 높음)
- 배당성향 20~30% (US 35~50% 대비 낮음)
- 중간배당 기업: 삼성전자, SK텔레콤, KB금융, POSCO 등 소수

## 2. Step 2: ENTRY (진입)

### 진입 점수 구성
1. **밸류에이션 (40%)**: PER 5~12배 + PBR 0.3~1.0배
2. **배당 품질 (30%)**: 배당수익률 + 성장 가산
3. **재무 건전성 (20%)**: ROE + 안정성
4. **타이밍 (10%)**: RSI + 배당락일 근접 패널티

### 등급 체계
- STRONG_BUY: ≥85점
- BUY: 70~84점
- HOLD: 55~69점
- PASS: <55점

## 3. Step 3: HOLD (보유)

### 모니터링 체크리스트
1. 배당 유지 여부 (감배 없음)
2. 배당성향 80% 이하 유지
3. 부채비율 150% 이하 유지
4. 영업이익 흑자 유지
5. 지배구조 이슈 없음
6. 시가총액 급락 없음 (-30% 이하)

### 보유 상태
- HEALTHY: 모든 조건 충족
- CAUTION: 1개 항목 미충족
- WARNING: 2개 이상 미충족
- EXIT_SIGNAL: 핵심 조건 위반

## 4. Step 4: COLLECT (배당 수령)

### 한국 배당 캘린더
- 기준일: 12월 31일 (대부분)
- 배당락일: 기준일 2영업일 전
- 지급일: 기준일 후 30~60일 (주총 후)
- 중간배당: 6월 30일 기준 (일부 기업)

### 수령 전략
- 배당락일 보유 유지 (홀드 스루)
- 배당락일 최소 2영업일 전 보유 필수
- 배당금 재투자 권장

## 5. Step 5: EXIT (매도)

### EXIT 트리거 (6가지)
1. **감배 발생** (HIGH) → 즉시 검토
2. **무배당 전환** (CRITICAL) → 매도 권고
3. **배당성향 100% 초과** (MEDIUM) → 경고
4. **2분기 연속 영업적자** (HIGH) → 검토
5. **부채비율 200% 초과** (MEDIUM) → 경고
6. **주가 -30% 급락** (HIGH) → 검토

### 매도 원칙
- 자동 매도 없음 (반드시 사용자 확인)
- 트리거 발생 → REVIEW 상태 → 사용자 판단
- 일시적 이벤트 vs 구조적 문제 구분
