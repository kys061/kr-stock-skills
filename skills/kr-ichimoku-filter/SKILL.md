# kr-ichimoku-filter: 일목균형표 전환선 필터

> 일봉 기준 종가가 일목균형표 전환선(9일) 위에서 마감했는지 확인하는 확인 필터.
> 스크리너 스킬(kr-pead-screener, kr-vcp-screener 등)에서 추가 확인 지표로 참조.

## 사용 시점

- 스크리너 스킬이 종목 추천 시 추가 확인 필터로 호출
- 단독 실행: 특정 종목의 일목균형표 전환선 상태 확인

## 방법론

### 전환선 (Tenkan-sen)
- **계산**: (9일 최고가 + 9일 최저가) / 2
- **판정**: 당일 종가 > 전환선 = PASS / 종가 <= 전환선 = FAIL

### 판정 기준

| 조건 | 결과 | 의미 |
|------|------|------|
| Close > Tenkan | PASS | 단기 상승 모멘텀 유지 |
| Close <= Tenkan | FAIL | 단기 모멘텀 약화 |

### 보조 정보 (참고용)

| 항목 | 설명 |
|------|------|
| 기준선 (Kijun) | 26일 고저 중값 - 중기 추세 |
| 구름대 (Kumo) | 선행스팬A/B 사이 - 지지/저항 |
| 전환선-기준선 관계 | 전환선 > 기준선 = 추세 강화 |
