# 한국 시장 분배일 가이드

## 1. 분배일 정의 (O'Neil 방법론)

### 기본 조건
1. **지수 종가** 전일 대비 **-0.2% 이상** 하락
2. **거래량**이 **전일 거래량 이상**
3. **25 거래일** 윈도우 내 발생

### 한국 시장 적용
- KOSPI + KOSDAQ **이중 추적** (US: S&P 500 + NASDAQ)
- 한국 시장은 프로그램 매매와 선물 옵션 만기의 영향이 큼
- 매월 둘째 목요일 (선물 옵션 동시 만기일) 거래량 왜곡 주의

## 2. 분배일 해석

### 누적 분배일별 의미

| 분배일 수 (25일 내) | 위험도 | 역사적 사례 |
|:------------------:|:------:|-----------|
| 0-2 | 정상 | 대부분의 상승기 |
| 3 | 초기 경고 | 조정 가능성 시작 |
| 4 | 주의 | 2018년 1월 (고점 형성 시작) |
| 5 | 위험 | 2021년 6-7월 (KOSPI 3,316 고점) |
| 6+ | 매우 위험 | 2007년 10-11월 (KOSPI 2,085 고점) |

### 분배일 클러스터링
- 5 거래일 내 2일 이상 분배일 → 1.15배 강화
- 연속 분배일 (2일 연속) → 매도 압력 극심

## 3. KOSPI/KOSDAQ 이중 추적

### 복합 점수 산출
```
최종 점수 = max(KOSPI 점수, KOSDAQ 점수) × 0.7 + min 점수 × 0.3
```

### 이중 신호 해석
- **두 지수 동시 분배일 누적**: 전면적 기관 매도 → 가장 위험
- **KOSPI만 높음**: 외국인/기관 대형주 매도 → 방어적 조정
- **KOSDAQ만 높음**: 개인 투자자 매도 → 투기 과열 해소

## 4. 한국 특수 요소

### 외국인 매매와 분배일
- 외국인이 KOSPI 거래의 ~25-30%를 차지
- 외국인 순매도 + 분배일 = 가장 강력한 천장 신호
- Component 7 (Foreign Flow)과 결합하여 해석

### 프로그램 매매 영향
- 차익 프로그램: 선물 기반 기계적 매매 → 분배일 판정에서 노이즈 가능
- 비차익 프로그램: 기관 전략 변경 → 유의미한 분배일 신호

### 옵션 만기 주의
- 매월 둘째 목요일 (선물 옵션 동시 만기)
- 분기 만기일 (3, 6, 9, 12월) 특히 거래량 급증
- 만기일 전후의 분배일은 신호 강도를 0.7배로 할인

## 5. 역사적 검증

### 2021년 6-7월 KOSPI 천장
- 6/25: 분배일 1 (KOSPI -0.4%, 거래량 증가)
- 6/29: 분배일 2 (KOSPI -0.3%)
- 7/6: 분배일 3 (KOSPI -0.7%)
- 7/8: 분배일 4 (KOSPI -0.5%)
- 7/13: 분배일 5 (KOSPI -0.9%, 외국인 대량 순매도)
→ 25일 내 5회 분배일 → 고위험 신호
→ KOSPI 3,316 → 2,608 (-21%, 12월까지)
