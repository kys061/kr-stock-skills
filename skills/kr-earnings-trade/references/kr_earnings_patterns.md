# 한국 실적 반응 패턴

## 한국 vs US 실적 반응 차이

| 특성 | 한국 | US |
|------|------|-----|
| 갭 크기 | ±30% 제한 | 무제한 |
| 서프라이즈 반영 | 잠정실적로 선반영 | 어닝콜에서 확인 |
| 기관 반응 속도 | 외국인/기관 일별 추적 가능 | 13F 분기 45일 지연 |
| 가이던스 | 거의 미제공 | 일반적 |
| 어닝콜 | 대형주 일부만 | 대부분 |

## 한국 실적 갭 패턴

### 긍정적 서프라이즈
1. 잠정실적 → 컨센서스 상회 → 갭업
2. 외국인 순매수 동반 시 추가 상승 가능성 높음
3. 기관 매수 + 개인 매도 = 강한 축적 신호

### 부정적 서프라이즈
1. 잠정실적 → 컨센서스 하회 → 갭다운
2. 외국인 순매도 → 추가 하락 가능성
3. ±30% 가격제한폭으로 1일 낙폭 제한

### 한국 특유 패턴
- **잠정→확정 수정**: 잠정실적과 확정실적 차이 발생 시 2차 갭
- **기대선반영**: 컨센서스가 이미 높으면 서프라이즈에도 갭 제한
- **수급 주도**: 외국인/기관 매매가 갭 후 방향 결정

## 등급별 투자 전략

### Grade A (85-100)
- 실적 갭 + 추세 + 거래량 + MA 모두 양호
- 즉시 진입 또는 첫 풀백 매수 고려
- 외국인 순매수 동반 시 확신도 상승

### Grade B (70-84)
- 모니터링 대상
- 추가 풀백 후 진입 기회 탐색
- MA50 지지 확인 후 진입

### Grade C (55-69)
- 추가 분석 필요
- 한 가지 이상 약한 팩터 존재
- 시장 환경 호조 시에만 소량 진입

### Grade D (0-54)
- 회피 권장
- 갭 크기/추세/거래량 중 다수 약함
