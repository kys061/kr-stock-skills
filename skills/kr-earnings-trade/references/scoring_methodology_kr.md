# 한국 실적 트레이드 5팩터 스코어링 방법론

## 개요
US earnings-trade-analyzer의 5팩터 스코어링 시스템을 한국 시장에 적용.
가중치와 스코어 테이블은 US와 동일하게 유지하되, 갭 계산 로직만 한국 특화.

## Factor 1: Gap Size (25%)
실적 발표 후 갭 크기. 한국은 ±30% 가격제한폭으로 자연 캡핑.

| 절대 갭 | 점수 |
|:-------:|:----:|
| ≥ 10% | 100 |
| ≥ 7% | 85 |
| ≥ 5% | 70 |
| ≥ 3% | 55 |
| ≥ 1% | 35 |
| < 1% | 15 |

### 한국 갭 계산
- 장전 공시 (08:00 이전): gap = open[D] / close[D-1] - 1
- 장중 공시 (08:00-15:30): gap = close[D] / open[D] - 1
- 장후 공시 (15:30 이후): gap = open[D+1] / close[D] - 1

## Factor 2: Pre-Earnings Trend (30%)
실적 전 20일 수익률. US와 동일.

| 20일 수익률 | 점수 |
|:----------:|:----:|
| ≥ 15% | 100 |
| ≥ 10% | 85 |
| ≥ 5% | 70 |
| ≥ 0% | 50 |
| ≥ -5% | 30 |
| < -5% | 15 |

## 외국인 순매수 보너스 (한국 특화)
