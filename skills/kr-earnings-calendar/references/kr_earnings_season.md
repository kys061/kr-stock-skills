# 한국 실적 시즌 가이드

## 연간 실적 발표 패턴

### 4Q 실적 (1-3월)
- **1-2월**: 잠정실적 공시 시작 (대형주 → 중소형주 순)
  - 삼성전자, SK하이닉스 등 대형주가 1월 초 공시
  - 중소형주는 2월까지 지속
- **3월 31일**: 사업보고서 제출 마감 (4Q 확정실적)
  - 감사 완료 재무제표 포함
  - 가장 중요한 공시 (연간 실적 확정)

### 1Q 실적 (4-5월)
- **4월**: 잠정실적 공시 시작
- **5월 15일**: 분기보고서 제출 마감 (1Q 확정실적)

### 2Q 실적 (7-8월)
- **7월**: 잠정실적 공시 시작
- **8월 14일**: 반기보고서 제출 마감 (2Q 확정실적)
  - 반기 감사 포함

### 3Q 실적 (10-11월)
- **10월**: 잠정실적 공시 시작
- **11월 14일**: 분기보고서 제출 마감 (3Q 확정실적)

## 잠정실적 vs 확정실적

| 항목 | 잠정실적 | 확정실적 |
|------|---------|---------|
| 공시 유형 | 영업(잠정)실적 (D002) | 정기보고서 (A001/A002/A003) |
| 포함 내용 | 매출액, 영업이익, 순이익 | 완전 재무제표 |
| 감사 여부 | 미감사 | 감사 완료 |
| 수정 가능 | 가능 (확정 시 수정 공시) | 확정 |
| 시장 반응 | 높음 (첫 실적 공개) | 보통 (이미 알려진 정보) |

## 한국 실적 발표 특성 (vs US)

| 특성 | 한국 | US |
|------|------|-----|
| 발표 시간 | 장중/장후 DART 공시 | BMO/AMC 구분 |
| 갭 크기 | ±30% 가격제한폭 | 제한 없음 |
| 컨퍼런스콜 | 일부 대형주만 | 대부분 실시 |
| 가이던스 | 거의 제공 안 함 | 일반적 |
| 실적 서프라이즈 | 잠정치로 선반영 | 어닝콜에서 확인 |

## 투자 전략 적용

1. **잠정실적 공시 전**: 컨센서스 대비 예상 확인
2. **잠정실적 공시 후**: 갭 방향 + 거래량으로 초기 반응 평가
3. **확정실적 공시**: 잠정치 대비 수정 여부 확인 (드물지만 중요)
4. **실적 시즌 외**: 수시 공시 (매출 변동, 수주 등) 모니터링
