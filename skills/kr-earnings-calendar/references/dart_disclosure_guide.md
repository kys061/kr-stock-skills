# DART 공시 유형/코드 가이드

## DART (전자공시시스템) 개요

금융감독원이 운영하는 전자공시시스템으로, 모든 상장법인의 공시 정보를 제공한다.
DART API를 통해 프로그래밍 방식으로 공시 정보를 조회할 수 있다.

## 실적 관련 공시 유형

### 정기보고서 (확정실적)

| 보고서 | DART 코드 | 주기 | 제출 마감 | 포함 내용 |
|--------|:---------:|:----:|----------|----------|
| 사업보고서 | A001 | 연 1회 | 결산일 후 90일 (3/31) | 연간 재무제표 + 감사보고서 |
| 반기보고서 | A002 | 연 1회 | 반기 종료 후 45일 (8/14) | 반기 재무제표 |
| 분기보고서 | A003 | 연 2회 | 분기 종료 후 45일 (5/15, 11/14) | 분기 재무제표 |

### 수시공시 (잠정실적)

| 공시 유형 | DART 코드 | 시점 | 내용 |
|----------|:---------:|------|------|
| 영업(잠정)실적 | D002 | 실적 시즌 | 매출액, 영업이익, 순이익 (속보치) |
| 매출액/손익구조 변경 | D001 | 수시 | 전 분기 대비 매출/손익 30% 이상 변동 시 |

## 공시 시간 분류

DART 공시는 시간 제한 없이 24시간 접수 가능하나, 일반적 패턴:

| 구분 | 시간대 | 시장 영향 |
|------|--------|----------|
| 장전 | ~08:00 | 시가에 반영 (갭) |
| 장중 | 08:00-15:30 | 실시간 주가 반응 |
| 장후 | 15:30~ | 익일 시가에 반영 |

## DART API 사용

### API 키 발급
- https://opendart.fss.or.kr 에서 발급
- 환경변수: `DART_API_KEY`

### 주요 API 엔드포인트
- 공시검색: `/api/list.json` (기업별/유형별 공시 목록)
- 기업개황: `/api/company.json` (기업 기본 정보)
- 재무제표: `/api/fnlttSinglAcntAll.json` (단일 회사 전체 재무제표)

### OpenDartReader 라이브러리
```python
import OpenDartReader as dart
api = dart.OpenDartReader('YOUR_API_KEY')

# 최근 공시 목록
disclosures = api.list(corp_code='005930', kind='D', start='2026-01-01')

# 재무제표
fs = api.finstate('005930', 2025)
```
