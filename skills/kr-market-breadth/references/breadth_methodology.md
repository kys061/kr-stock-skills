# 시장폭 분석 방법론

## 1. Breadth Index 정의

Breadth Index = (200일 이동평균 위 종목 수 / 전체 종목 수) × 100

- 범위: 0 ~ 100
- 50 이상: 과반수가 상승 추세 (건강)
- 50 이하: 과반수가 하락 추세 (주의)

## 2. 스무딩

- Raw: 당일 Breadth Index (노이즈 많음)
- 8MA: 8일 단순이동평균 (단기 추세)
- 200MA: 200일 단순이동평균 (장기 추세)

## 3. 6-컴포넌트 스코어링 상세

### 3.1 Breadth Level & Trend (25%)
8MA 레벨 기준 기본점수:
- 8MA >= 70: 기본 80점
- 8MA >= 50: 기본 60점
- 8MA >= 30: 기본 40점
- 8MA < 30: 기본 20점

보정:
- 200MA 상승추세: +10
- 8MA 상승 방향: +10

### 3.2 8MA vs 200MA Crossover (20%)
갭 = 8MA - 200MA
- 갭 > 0 AND 확대중: 90점
- 갭 > 0 AND 축소중: 60점
- 갭 < 0 AND 축소중 (수렴): 40점
- 갭 < 0 AND 확대중 (발산): 10점

### 3.3 Peak/Trough Cycle (20%)
- 최근 저점 이후 상승중: 80점
- 최근 고점 이후 하락중: 30점
- 고점 근처: 60점 (전환 경계)
- 저점 근처: 50점 (반등 가능)

### 3.4 Bearish Signal (15%)
- 약세 신호 없음: 100점
- 약세 신호 활성: 0점
- 약세 신호 조건: 8MA < 40 AND 200MA 하락 추세

### 3.5 Historical Percentile (10%)
- 현재 Raw Breadth의 전체 히스토리 내 백분위
- 점수 = 백분위 × 100

### 3.6 지수 Divergence (10%)
20일 + 60일 윈도우에서 지수 방향과 시장폭 방향 비교:
- 동일 방향: 80점 (확인)
- 시장폭 약화 + 지수 상승: 30점 (위험한 괴리)
- 시장폭 강화 + 지수 하락: 70점 (긍정 신호)
