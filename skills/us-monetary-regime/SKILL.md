# us-monetary-regime

US 통화정책 기조, 금리 방향성, 글로벌 유동성, **경제 펀더멘털**을 종합 분석하여 한국 주식시장 영향도를 오버레이 점수로 제공한다.

## Usage

```
/us-monetary-regime
```

## Modules

| Module | Function | Output |
|--------|----------|--------|
| fed_stance_analyzer | `analyze_fed_stance()` | Fed 기조 점수 (-100~+100) |
| rate_trend_classifier | `classify_rate_trend()` | 금리 5단계 분류 + 점수 (0~100) |
| liquidity_tracker | `track_liquidity()` | 유동성 점수 (0~100) |
| **economic_fundamentals_analyzer** | `analyze_fundamentals()` | **경제 펀더멘털 점수 (0~100)** |
| kr_transmission_scorer | `score_kr_transmission()` | 한국 전이 점수 + 오버레이 (+-15) |
| regime_synthesizer | `synthesize_regime()` | 종합 레짐 + 섹터별 오버레이 |

## Economic Fundamentals (Fed 금리 결정의 근본 원인)

Fed의 Dual Mandate(물가 안정 + 최대 고용)와 경기 상태를 분석하여 금리 인상/인하 압력을 예측한다.

### 4-컴포넌트

| 컴포넌트 | 가중치 | 핵심 지표 | Fed 결정 영향 |
|---------|:------:|----------|-------------|
| **인플레이션** | 35% | CPI YoY, Core PCE, 방향성 | 목표 2% 대비 → 인상/인하 압력 |
| **고용/노동시장** | 30% | 실업률, 비농업고용(NFP), 임금상승률 | 과열→인상, 냉각→인하 |
| **경기 성장** | 25% | GDP, ISM 제조업/서비스, 선행지표(LEI) | 침체→인하, 과열→인상 |
| **외생적 충격** | 10% | 지정학/팬데믹/금융위기/무역전쟁 | 위기→긴급 인하 |

### 인플레이션 분석
- CPI YoY 또는 Core PCE (Fed 선호 지표) 사용
- 방향성: accelerating / stable / decelerating / falling
- 목표 2.0% 대비 갭 계산 → 인상/인하 압력 판단

### 고용 분석
- 실업률 vs 균형 범위 (3.5~4.5%)
- 비농업고용(NFP): 100~200K = 적정
- 임금상승률: 3~4% = 정상, 5%+ = 임금-물가 스파이럴 우려

### 경기 성장 분석
- GDP 연율화: 2~3% = 적정 성장
- ISM 제조업/서비스: 50 기준 (확장/수축)
- 선행경제지표(LEI): 6M 변화율로 침체 선행 탐지

### 외생적 충격 분석
- 충격 수준: none / minor / moderate / severe / crisis
- 유형: 팬데믹, 전쟁, 금융위기, 무역전쟁, 에너지위기, 공급망 등
- **인플레이션 유발 여부** 구분:
  - 공급 충격(유가↑, 관세↑) → 인하 제약 (스태그플레이션 위험)
  - 수요 충격(팬데믹, 금융위기) → 인하 촉진
- 지속 기간: 6M+ 장기 충격은 추가 영향

### Pressure Labels
| 라벨 | 점수 | 의미 |

## Integration (B방식 Overlay)

기존 kr-스킬의 점수 체계를 변경하지 않고, 오버레이로 적용:

```python
# kr-stock-analysis
from comprehensive_scorer import apply_monetary_overlay
result = apply_monetary_overlay(base_score=60.0, overlay=8.5, sector='semiconductor')
# → 60.0 + 8.5 * 1.3 = 71.05

# overlay=None이면 기존과 동일
result = apply_monetary_overlay(base_score=60.0, overlay=None)
# → 60.0
```

## Scoring

### US Regime Score (0~100)
- stance × 0.25 + rate × 0.20 + liquidity × 0.25 + **fundamentals × 0.30**
- 0~35: Tightening / 35~65: Hold / 65~100: Easing

### WebSearch 필수 조회 항목
분석 실행 시 반드시 아래 데이터를 WebSearch로 수집한다:
1. Fed 기준금리, FOMC 최신 성명서 기조
2. **CPI YoY, Core PCE YoY, 인플레이션 방향**
3. **실업률, 최근 비농업고용(NFP), 임금상승률**
4. **GDP 성장률, ISM PMI**
5. DXY, 한미 금리차, BOK 기준금리
6. **현재 진행 중인 외생적 충격 (전쟁, 무역분쟁, 팬데믹 등)**

### Overlay Calculation
- `(regime_score - 50) * 0.30` = base overlay (+-15)
- `base_overlay * sector_sensitivity` = final overlay per sector

## Output Rule (마크다운 리포트 저장)

- 분석 완료 후 결과를 **마크다운 파일로 저장**한다
- 저장 경로: `reports/{skill_name}_{identifier}_{name}_{YYYYMMDD}.md`
  - `{identifier}`: 종목코드 (없으면 `market`)
  - `{name}`: 종목명 또는 분석 대상명 (한글 가능)
  - 예: `reports/kr-stock-analysis_005930_삼성전자_20260305.md`
- 파일 구조:
  ```markdown
  # {스킬명} 분석 리포트
  > 생성일: {YYYY-MM-DD HH:MM} | 대상: {identifier}

  ## 요약
  (핵심 결론 2-3줄)

  ## 상세 분석
  (스킬별 분석 결과를 마크다운 테이블, 리스트, 헤더로 구조화)

  ## 판단 근거
  (점수, 등급, 시그널 등 정량 데이터)

  ---
  *Generated by {skill_name}*
  ```
- `reports/` 디렉토리가 없으면 자동 생성
- 동일 파일명이 존재하면 덮어쓰기 (같은 날 재분석 시)
