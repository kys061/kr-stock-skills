# KOSPI200 옵션 가이드

## 1. 상품 개요

| 항목 | 내용 |
|------|------|
| 기초자산 | KOSPI200 지수 |
| 거래 승수 | 250,000원 (1포인트 = 25만원) |
| 호가 단위 | 프리미엄 < 3pt: 0.01pt / ≥ 3pt: 0.05pt |
| 거래시간 | 09:00-15:45 (만기일 15:30) |
| 최종결제 | 현금결제 (T+1) |
| 만기일 | 매월 두 번째 목요일 |

## 2. 증거금 체계

- 매도 시 증거금 필요 (스프레드 매도는 순위험 기준)
- 매수 시 프리미엄 전액 지불
- SPAN 방식 증거금 계산

## 3. VKOSPI

- 한국판 VIX — KOSPI200 옵션의 내재 변동성 지수
- 일반 범위: 12-25
- 위기 시: 40+
- 평균: ~18-22

## 4. Black-Scholes 모델

### 콜 옵션 가격
C = S·N(d₁) - K·e^(-rT)·N(d₂)

### 풋 옵션 가격
P = K·e^(-rT)·N(-d₂) - S·N(-d₁)

### d₁, d₂
d₁ = [ln(S/K) + (r + σ²/2)·T] / (σ·√T)
d₂ = d₁ - σ·√T

## 5. 그릭스 (Greeks)

| 그릭 | 의미 | 콜 범위 | 풋 범위 |
|------|------|---------|---------|
| Delta (δ) | 기초자산 1pt 변동 시 옵션가 변화 | 0~1 | -1~0 |
| Gamma (γ) | 델타의 변화율 | 양수 | 양수 |
| Theta (θ) | 시간가치 일일 감소 | 음수 | 음수 |
| Vega (ν) | 변동성 1% 변동 시 옵션가 변화 | 양수 | 양수 |
| Rho (ρ) | 금리 1% 변동 시 옵션가 변화 | 양수 | 음수 |

## 6. ATM/ITM/OTM 판정

- ATM: 행사가가 현재가의 ±2% 이내
- ITM: 콜(S>K), 풋(S<K)
- OTM: 콜(S<K), 풋(S>K)
