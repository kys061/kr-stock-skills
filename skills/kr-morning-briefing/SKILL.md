# kr-morning-briefing: 장 초반 브리핑

> KST 08:00~09:00 실행. 글로벌 시장 현황(27개) + 당월 주요 일정 + 핫 키워드 2-3개를
> 하나의 리포트로 제공하는 장 초반 종합 브리핑 스킬.

## 사용법

```
/kr-morning-briefing
```

## 실행 절차

### Step 1: 글로벌 시장 데이터 수집 (자동)

`market_data_collector.py`가 yfinance 배치 다운로드로 17개 항목을 자동 수집한다.

### Step 2: WebSearch 시장 데이터 (10개)

아래 10개 항목은 yfinance 미지원이므로 WebSearch로 수집한다:

| 항목 | 검색 쿼리 |
|------|----------|
| 국고채 2년 | "미국 국채 2년 수익률 오늘" |
| 두바이유 | "두바이유 가격 오늘" |
| 알루미늄 | "LME 알루미늄 가격" |
| 니켈 | "LME 니켈 가격" |
| 철광석 | "철광석 가격 오늘" |
| 유연탄 | "호주 유연탄 가격 Newcastle" |
| 리튬 | "리튬 카보네이트 가격" |
| 쌀 | "쌀 국제 가격 CBOT" |
| SCFI | "SCFI 상하이 컨테이너 운임지수 최신" |
| BDI | "BDI 발틱운임지수 오늘" |

수집된 데이터를 `websearch_market` dict로 전달:
```python
websearch_market = {
    'us2y': {'price': 4.25, 'change_pct': -0.02},
    'dubai_oil': {'price': 72.5, 'change_pct': 1.2},
    ...
}
```

### Step 3: WebSearch 당월 일정

"한국 증시 {월}월 주요 일정 2026" 검색하여 동적 일정을 수집한다.
정적 일정(FOMC, BOK, 만기일 등)은 `references/monthly_events.json`에서 자동 로드.

### Step 4: WebSearch 핫 키워드

아래 쿼리로 장 초반 핫 이슈를 수집한다:
1. "한국 증시 오늘 장 전 핫 키워드"
2. "코스피 오늘 전망 {날짜}"
3. 전일 미국 시장 주요 뉴스

수집 결과를 분석하여 2-3개 핫 키워드 + 관련주 + 한줄평을 구성한다.

### Step 5: 리포트 생성

3개 Section을 조합하여 리포트를 생성한다:
- Section 1: 글로벌 시장 현황 (8개 카테고리)
- Section 2: 당월 주요 일정 체크리스트
- Section 3: 장초반 핫 키워드 + 관련주 + 한줄평

### Step 6: 저장 + 이메일 발송

```bash
cd ~/stock && python3 skills/_kr_common/utils/email_sender.py \
  "reports/kr-morning-briefing_market_장초반브리핑_{YYYYMMDD}.md" \
  "kr-morning-briefing"
```
