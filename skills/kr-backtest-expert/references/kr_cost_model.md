|------|:----:|------|

## Round-Trip 비용 계산

```
매수: 0.015% (수수료)
매도: 0.015% (수수료) + 0.23% (거래세)
슬리피지: 0.1% × 2
─────────────
합계: 0.46% + 0.2% = 0.66% (실질 비용)
```
