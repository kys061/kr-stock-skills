# 한국 시장 경기 순환과 업종 로테이션

## 1. 한국 경기 순환 특징

### 1.1 수출 주도 경제
- GDP의 40%+ 수출 의존
- 반도체/자동차/조선이 수출 3대 축
- 글로벌 경기와 높은 상관관계

### 1.2 반도체 슈퍼사이클
- 삼성전자 + SK하이닉스 = KOSPI 시총 약 30%
- 반도체 업황이 KOSPI 방향 좌우
- 메모리 반도체 가격 사이클: 2-3년 주기

### 1.3 외국인 수급의 결정적 영향
- KOSPI 외국인 지분율 30%+
- 외국인 순매수/순매도가 시장 방향성 결정
- MSCI 리밸런싱 이벤트 시 대규모 수급 변동

### 1.4 원달러 환율 민감도
- 원 강세 (환율 하락): 내수주 ↑, 수출주 ↓ (원화 수익 감소)
- 원 약세 (환율 상승): 수출주 ↑ (원화 수익 증가), 외국인 이탈 우려
- 환율-KOSPI 역상관 관계 (대체로)

## 2. 경기 국면별 선도 업종

### 2.1 초기 회복 (Early Recovery)

**특징**: 금리 인하, 경기 바닥 확인, 외국인 순매수 재개

| 선도 업종 | 근거 |
|-----------|------|
| 반도체 | 재고 사이클 바닥 탈출 |
| 자동차 | 소비 회복 수혜 |
| 건설 | 정부 부양 정책 수혜 |
| 증권 | 거래량 증가 수혜 |

**후행 업종**: 통신, 유틸리티 (방어주에서 자금 이탈)

### 2.2 중기 확장 (Mid-Cycle Expansion)

**특징**: 기업 실적 개선, 광범위한 참여, 원화 강세

| 선도 업종 | 근거 |
|-----------|------|
| IT/2차전지 | 성장 프리미엄 |
| 금융 (은행) | 순이자마진 개선 |
| 화학 | 경기 민감 |
| 전반적 상승 | 시장 참여 폭 확대 |

### 2.3 후기 과열 (Late Cycle)

**특징**: 금리 인상 우려, 원자재 가격 상승, 레버리지 확대

| 선도 업종 | 근거 |
|-----------|------|
| 조선 | 장기 수주 사이클 |
| 정유/에너지 | 원자재 가격 상승 |
| 은행 | 금리 상승 수혜 |
| 필수소비재 | 디펜시브 전환 시작 |

**후행 업종**: 성장주 (IT, 2차전지), 바이오

### 2.4 침체 (Recession)

**특징**: 수출 감소, 외국인 순매도, 원화 약세

| 선도 업종 | 근거 |
|-----------|------|
| 통신 | 안정적 현금흐름, 배당 |
| 유틸리티 | 필수재 수요 |
| 필수소비재 | 비탄력적 수요 |
| 제약/바이오 | 비경기민감 |

## 3. 한국 특유의 로테이션 패턴

### 3.1 반도체 파급 효과
```
반도체 업사이클 시작
  → IT부품/장비 (2-3개월 후행)
  → PCB/패키징 (3-6개월 후행)
  → 소재/화학 (6개월+ 후행)
```

### 3.2 사이즈 로테이션
```
외국인 순매수 시작
  → 대형주 (삼성전자, SK하이닉스)
  → 중형주 (업종 대표주)
  → 소형주 (테마/모멘텀)
```

### 3.3 환율 기반 로테이션
- 환율 하락 (원 강세): 내수주 상승 (유통, 통신, 음식료)
- 환율 상승 (원 약세): 수출주 상승 (반도체, 자동차, 조선)

### 3.4 밸류업 로테이션 (2024~)
- 정부 주도 기업가치 제고 프로그램
- 저PBR 금융/지주주 재평가
- 배당 확대 및 자사주 소각 기대
