# 2026년 한국 투자 세제 코드 레퍼런스

## 1. 배당소득세

- **세율**: 15.4% (소득세 14% + 지방소득세 1.4%)
- **과세 방식**: 원천징수 (지급 시 자동 공제)
- **비과세 배당**: 없음 (ISA 계좌 통해 간접 비과세 가능)

## 2. 금융소득종합과세

- **기준**: 연간 금융소득(이자+배당) 합계 2,000만원 초과 시
- **과세 방법**: 2,000만원까지 15.4% 원천징수, 초과분 종합소득세율 적용
- **종합소득세율** (지방세 별도):

| 구간 | 세율 |
|------|:----:|
| ~1,200만원 | 6% |
| ~4,600만원 | 15% |
| ~8,800만원 | 24% |
| ~1.5억원 | 35% |
| ~3억원 | 38% |
| ~5억원 | 40% |
| ~10억원 | 42% |
| 10억원 초과 | 45% |

- **지방소득세**: 소득세의 10% 추가

## 3. 양도소득세 (주식)

### 대주주 기준
- **대주주 판정**: 종목당 10억원 이상 보유
- **세율**: 1년 이상 보유 22%, 1년 미만 33% (지방세 포함)
- **중소기업**: 11% (지방세 포함)

### 소액주주 (금투세 시행 시)
- **세율**: 22%
- **기본공제**: 5,000만원

### 해외주식
- **세율**: 22%
- **기본공제**: 250만원/년

## 5. ISA (개인종합자산관리계좌)

- **비과세 한도**: 200만원 (서민형 400만원)
- **초과분**: 9.9% 분리과세
- **연간 납입한도**: 2,000만원
- **의무가입기간**: 3년
- **투자 가능**: 국내 주식, ETF, 펀드, ELW, RP

## 6. 연금저축

- **연간 납입한도**: 1,800만원
- **세액공제 한도**: 600만원
- **세액공제율**:
  - 총급여 5,500만원 이하: 16.5%
  - 총급여 5,500만원 초과: 13.2%
- **55세 미만 중도인출**: 16.5% 기타소득세
- **55세 이후 연금수령**:
  - 연 1,200만원 이하: 5.5%
  - 70세 이하: 5.5%
  - 80세 이하: 4.4%
  - 80세 초과: 3.3%

## 7. IRP (개인형 퇴직연금)

- **연간 납입한도**: 1,800만원 (연금저축 합산)
- **세액공제 한도**: 900만원 (연금저축 600 + IRP 300)
- **세액공제율**: 연금저축과 동일
- **안전자산 30% 의무**: 위험자산 70% 한도
- **투자 가능**: 펀드, ETF, 예금, RP, ELW 등
