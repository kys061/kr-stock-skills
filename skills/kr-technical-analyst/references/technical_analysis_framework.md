# 기술적 분석 프레임워크

## 1. 추세 분석

### 1.1 추세 정의
- **상승추세**: Higher High + Higher Low 지속
- **하락추세**: Lower High + Lower Low 지속
- **횡보**: 일정 범위 내 등락 반복

### 1.2 추세 강도 판정
| 강도 | 조건 |
|------|------|
| 강 | MA 정배열 + 거래량 확인 + 명확한 HH/HL |
| 중 | MA 일부 정렬 + 거래량 혼재 |
| 약 | MA 수렴 + 추세 둔화 + 거래량 감소 |

### 1.3 탈진 신호
- 거래량 클라이맥스 (급증 후 가격 정체)
- 기울기 급경사 후 둔화
- 장대양봉/장대음봉 후 반전 캔들

---

## 2. 지지/저항 분석

### 2.1 수평 S/R
- 이전 고점/저점에서 형성
- 접촉 횟수가 많을수록 강함
- 돌파 후 역할 전환 (저항→지지 또는 반대)

### 2.2 동적 S/R (이동평균)
- 20MA: 단기 동적 지지/저항
- 50MA: 중기 동적 지지/저항
- 200MA: 장기 추세 경계

### 2.3 컨플루언스 존
복수 요소가 수렴하는 가격대:
- 수평 S/R + MA + 피보나치 + 추세선
- 컨플루언스가 강할수록 반응 확률 높음

---

## 3. 이동평균 해석

### 3.1 정렬 상태
| 상태 | 의미 | 조건 |
|------|------|------|
| 정배열 (Bullish) | 상승 추세 | 가격 > 20MA > 50MA > 200MA |
| 역배열 (Bearish) | 하락 추세 | 가격 < 20MA < 50MA < 200MA |
| 수렴 | 추세 전환 가능 | MA간 거리 좁아짐 |

### 3.2 크로스오버
- **골든크로스**: 50MA가 200MA 상향 돌파 (장기 상승 전환)
- **데드크로스**: 50MA가 200MA 하향 돌파 (장기 하락 전환)
- 신뢰도: 거래량 확인 시 높아짐

---

## 4. 거래량 분석

### 4.1 기본 원칙
- 추세 방향 거래량 증가 = 추세 확인
- 추세 방향 거래량 감소 = 추세 약화
- 반대 방향 거래량 급증 = 전환 경고

### 4.2 특수 패턴
| 패턴 | 의미 |
|------|------|
| 거래량 클라이맥스 | 추세 반전 가능 |
| 건조 거래량 | 돌파 직전 가능 |
| 하락 시 거래량 감소 | 매도 탈진 |

---

## 5. 주요 차트 패턴

### 5.1 반전 패턴
- **해머/역해머**: 하락 후 반전 신호
- **유성형**: 상승 후 반전 신호
- **장악형 (Engulfing)**: 이전 캔들을 삼킴
- **이중천정/이중바닥**: M/W 패턴
- **헤드앤숄더**: 삼봉 반전 패턴

### 5.2 지속 패턴
- **깃발 (Flag)**: 급등/급락 후 좁은 통합
- **삼각형 (Triangle)**: 수렴 후 돌파 방향으로 지속
- **쐐기 (Wedge)**: 기울어진 수렴

### 5.3 한국 시장 특수 패턴
- **갭상한가**: 다음 날 추가 갭업 가능하나 괴리 확대 시 눌림 주의
- **외국인 매집 구간**: 점진적 상승 + 꾸준한 거래량 = 외국인 순매수 가능성
- **프로그램 매도 영향**: 장 후반 급락 = 프로그램 매도 가능성

## 6. 시나리오 개발 가이드

### 6.1 확률 배정 원칙
- 모든 시나리오 합계 = ~100%
- Base Case가 40-60% 차지
- 최소 2개, 최대 4개 시나리오
- 각 시나리오는 상호 배타적

### 6.2 시나리오 품질 기준
- 기술적 근거 2-3개 이상 제시
- 구체적 목표가 (모호한 "상승" 금지)
- 명확한 무효화 레벨
- 근거 강도에 비례한 확률

### 6.3 한국 시장 보정 요소
| 보조 데이터 | 보정 방향 | 범위 |
|-------------|-----------|------|
| 외국인 연속 순매수 10일+ | Bull +5~10% | 수급 지지 |
| 기관 대규모 순매도 | Bear +5~10% | 수급 악화 |
| 공매도 잔고 급증 | Bear +5% | 하방 압력 |
| 신용잔고 급증 | Caution | 과열 징후 |
