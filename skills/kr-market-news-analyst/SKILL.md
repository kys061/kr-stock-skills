# kr-market-news-analyst

> 한국 시장 뉴스 분석 스킬 - 최근 10일간 시장 영향 뉴스의 임팩트 스코어링 및 분석 리포트

## 개요

최근 10일간 한국 주식 시장에 영향을 미친 주요 뉴스를 자동 수집하고,
임팩트 스코어를 부여하여 중요도 순위 분석 리포트를 생성한다.

- **데이터 소스**: WebSearch + WebFetch (한국/글로벌 뉴스)
- **API 불필요**: WebSearch로 자동 수집
- **출력**: Markdown 분석 리포트

## 실행 조건

사용자가 최근 시장 뉴스 분석, 주요 이벤트 영향 분석 등을 요청할 때 사용한다.

## 실행 워크플로우

### Step 1: 뉴스 수집 (6개 카테고리)

WebSearch로 아래 카테고리별 최근 10일간 주요 뉴스 수집:

1. **한은 금통위 / 통화정책**
   - 검색: "한은 기준금리 결정" "한국은행 금통위"
   - 글로벌: "FOMC rate decision" "Fed policy"

2. **경제지표**
   - 검색: "한국 CPI 소비자물가" "한국 수출 증감" "한국 GDP"
   - 글로벌: "US CPI" "nonfarm payrolls" "US GDP"

3. **대형주 실적**
   - 검색: "삼성전자 실적" "SK하이닉스 실적" "현대차 실적"
   - 글로벌: "NVIDIA earnings" "Apple earnings"

4. **외국인 수급 이벤트**
   - 검색: "외국인 순매수 순매도" "MSCI 리밸런싱"

5. **정책/규제**
   - 검색: "공매도 정책" "밸류업 프로그램" "금융위원회"

6. **지정학/글로벌 리스크**
   - 검색: "미중 갈등 반도체" "한반도 지정학"
   - 검색: "원달러 환율 급변"

### Step 2: 레퍼런스 로드

```
Read references/market_event_patterns.md
Read references/trusted_kr_news_sources.md
Read references/kr_market_correlations.md
```

### Step 3: 임팩트 스코어 계산

```
Impact Score = (Price Impact × Breadth Multiplier) × Forward Modifier

Price Impact Score (한국 시장 기준):
┌──────────┬───────┬──────────────────────────┐
│ 등급     │ 점수  │ 기준                     │
├──────────┼───────┼──────────────────────────┤
│ Severe   │ 10점  │ KOSPI ±2%+, 원달러 ±2%+  │
│ Major    │  7점  │ KOSPI ±1~2%, 원달러 ±1~2% │
│ Moderate │  4점  │ KOSPI ±0.5~1%            │
│ Minor    │  2점  │ KOSPI ±0.2~0.5%          │
│ Negligible│ 1점  │ <0.2%                    │
└──────────┴───────┴──────────────────────────┘

Breadth Multiplier:
- Systemic: 3x (복수 자산/글로벌 영향)
- Cross-Asset: 2x (주식 + 환율/채권)
- Sector-Wide: 1.5x (업종 전체)
- Stock-Specific: 1x (개별 종목)

Forward Modifier:
- 정책 레짐 전환: +50% (금리 인하/인상 전환, 공매도 정책 변경)
- 외국인 수급 전환: +30% (연속 순매수 → 순매도 전환)
- 추세 확인: +25%
- 단발 이벤트: 0%
- 반대 신호: -25%
```

### Step 4: 이벤트 반응 분석

각 이벤트에 대해:
1. **즉시 반응** (당일): KOSPI/KOSDAQ/환율 변동
2. **후속 반응** (2-3일): 추세 지속 여부
3. **패턴 비교**: 예상 vs 실제 반응
4. **자산별 영향**: 주식/채권/환율/원자재

### Step 5: 리포트 생성

## 뉴스 소스 우선순위

### Tier 1: 공식 소스
- 한국은행 (bok.or.kr) - 금통위, 기준금리
- 금융위원회 (fsc.go.kr) - 금융 규제/정책
- 통계청 (kostat.go.kr) - 경제지표
- DART (dart.fss.or.kr) - 기업 공시
- Federal Reserve, SEC, BLS - 미국 공식

### Tier 2: 경제 전문지
- 한국경제 (hankyung.com)
- 매일경제 (mk.co.kr)
- 연합인포맥스 (einfomax.co.kr)
- 조선비즈 (biz.chosun.com)
- Bloomberg, Reuters, CNBC

### Tier 3: 증권사 리서치
- 키움증권, 삼성증권, NH투자증권

## 한국 시장 이벤트 카테고리

### 1. 한은 금통위 (연 8회)
- 기준금리 결정
- 시장 영향: 즉시 (채권 > 환율 > 주식)
- 패턴: 동결 = 제한적 영향, 인상/인하 = Major+

### 2. 경제지표
- 수출 증감 (매월 1일) - 한국 핵심 선행지표
- CPI, GDP (분기)
- 고용

### 3. 대형주 실적
- 삼성전자 (KOSPI 시총 20%+): 실적이 시장 방향 좌우
- SK하이닉스: 반도체 사이클 바로미터
- 현대차/기아: 수출 및 자동차 업황
- LG에너지솔루션: 2차전지 센티먼트

### 4. 외국인 수급 이벤트
- MSCI 리밸런싱 (연 4회): 편입/제외 종목 수급 급변
- 대규모 순매수/순매도 전환일

### 5. 정책/규제
- 공매도 정책 변경
- 밸류업 프로그램 발표
- 세제 변경 (금투세, 양도세)

### 6. 지정학 리스크
- 미중 반도체 갈등 → 삼성/하이닉스 영향
- 한반도 지정학 → 외국인 이탈 트리거

## 분석 원칙

1. **인과관계 엄격**: 상관관계 ≠ 인과관계, 추론 시 명시적 표기
2. **복수 자산 관점**: 주식만이 아닌 채권/환율/원자재 통합 분석
3. **이상 반응 식별**: 예상과 다른 시장 반응에 주목
4. **한국어 우선**: 한국 시장 뉴스는 한국어로 분석, 글로벌은 영어 소스 참조
5. **정량 기반**: 임팩트 스코어로 객관적 순위화

## Output Rule (마크다운 리포트 저장)

- 분석 완료 후 결과를 **마크다운 파일로 저장**한다
- 저장 경로: `reports/{skill_name}_{identifier}_{name}_{YYYYMMDD}.md`
  - `{identifier}`: 종목코드 (없으면 `market`)
  - `{name}`: 종목명 또는 분석 대상명 (한글 가능)
  - 예: `reports/kr-stock-analysis_005930_삼성전자_20260305.md`
- 파일 구조:
  ```markdown
  # {스킬명} 분석 리포트
  > 생성일: {YYYY-MM-DD HH:MM} | 대상: {identifier}

  ## 요약
  (핵심 결론 2-3줄)

  ## 상세 분석
  (스킬별 분석 결과를 마크다운 테이블, 리스트, 헤더로 구조화)

  ## 판단 근거
  (점수, 등급, 시그널 등 정량 데이터)

  ---
  *Generated by {skill_name}*
  ```
- `reports/` 디렉토리가 없으면 자동 생성
- 동일 파일명이 존재하면 덮어쓰기 (같은 날 재분석 시)
