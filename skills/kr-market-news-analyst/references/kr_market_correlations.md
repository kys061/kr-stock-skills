# 한국 시장 상관관계 프레임워크

## 1. 인터마켓 상관관계

### KOSPI ↔ 글로벌 지수
| 지수 | 상관도 | 메커니즘 |
|------|:------:|----------|
| S&P 500 | 높음 | 글로벌 리스크 센티먼트 |
| NASDAQ | 높음 | 기술주 센티먼트 (반도체) |
| 니케이 225 | 중간 | 아시아 리스크, 경쟁 관계 |
| 상해종합 | 중간 | 중국 경기 → 한국 수출 |
| VIX | 역상관 | 공포 지수 상승 = KOSPI 하락 |

### KOSPI ↔ 환율
- USD/KRW ↑ (원 약세) ↔ KOSPI ↓ (역상관, 상관계수 -0.6~-0.8)
- 메커니즘: 원 약세 → 외국인 자산 감소 → 매도 → KOSPI 하락

### KOSPI ↔ 원자재
| 원자재 | 영향 |
|--------|------|
| 유가 ↑ | 정유↑, 운송↓, 화학 혼재 |
| 구리 ↑ | 경기 회복 신호, 수출주↑ |
| 금 ↑ | 안전자산 선호, KOSPI 약세 가능 |

## 2. 섹터 간 상관관계

### 반도체 파급
```
삼성전자/하이닉스 상승
  → IT 부품/장비 (2-3일 후행)
  → PCB/패키징 (1주 후행)
  → 소재/화학 (2주+ 후행)
```

### 원화 환율 영향
```
원 강세 (환율 하락)
  → 내수주 ↑ (유통, 통신, 음식료)
  → 수출주 ↓ (반도체, 자동차, 조선)

원 약세 (환율 상승)
  → 수출주 ↑
  → 내수주 ↓
```

### 금리 영향
```
금리 상승
  → 은행 ↑ (순이자마진 확대)
  → 건설/부동산 ↓ (이자 비용 증가)
  → 성장주 ↓ (할인율 상승)

금리 하락
  → 성장주 ↑ (할인율 하락)
  → 건설/부동산 ↑
  → 은행 ↓ (순이자마진 축소)
```
