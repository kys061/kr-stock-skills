# DART 공시 유형 레퍼런스

## 1. 공시 종류 (kind)

| 코드 | 유형 | 설명 | 모니터링 대상 |
|:----:|------|------|:------------:|
| A | 정기보고서 | 사업보고서/반기/분기보고서 | T3 실적 |
| B | 주요사항보고서 | 배당결정/유무상증자/합병 등 | T1/T2 배당 |
| C | 발행공시 | 증권 발행 관련 | - |
| D | 지분공시 | 5% 대량보유/임원 지분변동 | T5 지배구조 |
| E | 기타 | 기타 공시 | - |

## 3. 배당 관련 키워드

- 배당, 현금배당, 주당배당금
- 배당기준일, 배당락일
- 무배당, 배당 미결의
- 중간배당, 분기배당

## 4. 지배구조 관련 키워드

- 대량보유, 5% 보고
- 임원 지분변동
- 감사의견, 적정, 한정, 부적정
- 관리종목, 상장폐지
