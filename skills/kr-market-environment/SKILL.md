# kr-market-environment

> 한국 시장 환경 분석 스킬 - 글로벌+한국 시장 종합 진단 리포트 생성

## 개요

한국 시장을 중심으로 글로벌 시장 환경을 종합 분석하고, 전문 시장 환경 리포트를 생성한다.
WebSearch로 실시간 글로벌 데이터를 수집하고, KRClient로 한국 시장 정량 데이터를 보강한다.

- **데이터 소스**: WebSearch (글로벌) + KRClient (한국 정량 데이터)
- **스크립트**: `scripts/market_utils.py` (한국 시장 스냅샷)
- **출력**: Markdown 분석 리포트

## 실행 조건

사용자가 시장 환경 분석, 시장 현황, 오늘 시장 상태 등을 요청할 때 사용한다.

## 분석 프레임워크 (6차원)

### 기본 4차원 (US 원본 보존)
1. **Trend Direction**: 상승추세 / 하락추세 / 박스권
2. **Risk Sentiment**: Risk-on / Risk-off
3. **Volatility**: VKOSPI + VIX 기반 시장 불안도
4. **Sector Rotation**: 자금 흐름 방향

### 한국 특화 2차원 (신규)
5. **수급 동향**: 기관/외국인 순매수 추세 (연속일수, 금액 규모)
6. **밸류에이션 위치**: KOSPI PER 밴드 내 현재 위치 (저평가/적정/고평가)

## 실행 워크플로우

### Step 1: 한국 시장 정량 데이터 수집

```bash
python3 skills/kr-market-environment/scripts/market_utils.py
```

출력: JSON 형태의 한국 시장 스냅샷
- KOSPI/KOSDAQ 현재가, 등락률, PER/PBR
- 원달러 환율
- 한국 국고채 수익률 (3Y/5Y/10Y)
- 기관/외국인 순매수 (최근 5일)
- VKOSPI
- KOSPI PER 밴드 위치

### Step 2: 글로벌 시장 데이터 수집 (WebSearch)

**필수 검색 항목**:
1. S&P 500, NASDAQ, Dow Jones 현재 수준 및 등락
2. VIX 지수
3. 아시아: 니케이 225, 상해종합, 항셍
4. 유럽: DAX, FTSE, CAC 40
5. 환율: USD/KRW, USD/JPY, EUR/USD
6. 원자재: WTI 유가, 금
7. 미국 국채: 2년/10년 수익률

**검색 쿼리 예시**:
- "KOSPI KOSDAQ 오늘 시장 현황"
- "S&P 500 NASDAQ market today"
- "USD KRW exchange rate today"
- "VIX index today"

### Step 3: 레퍼런스 로드

```
Read references/indicators.md
Read references/analysis_patterns.md
```

### Step 4: 6차원 분석

각 차원별 판정:
1. Trend Direction → 상승/하락/박스권
2. Risk Sentiment → Risk-on / Risk-off
3. Volatility → categorize_vkospi() 결과
4. Sector Rotation → 업종 자금 흐름 방향
5. 수급 동향 → 외국인/기관 순매수 연속일수, 금액
6. 밸류에이션 → PER 밴드 위치 (저평가/적정/고평가/과열)
