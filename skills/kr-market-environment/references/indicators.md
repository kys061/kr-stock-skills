# 한국 시장 주요 지표 해석 가이드

## 1. VKOSPI (한국판 VIX)

| 레벨 | 상태 | 의미 |
|:----:|------|------|
| <10 | 극도 안정 | 과도한 안도감, 역발상 경고 |
| 10-15 | 안정 | 정상적 시장 환경 |
| 15-20 | 보통 | 약간의 불안 |
| 20-25 | 불안 | 시장 긴장 |
| 25-30 | 경계 | 상당한 불확실성 |
| 30+ | 공포 | 패닉 매도 가능, 역발상 매수 기회 |

## 2. KOSPI PER 밴드

| 위치 | PER 범위 (최근 10년 기준) | 판정 |
|------|:------------------------:|------|
| 하단 | 8-10배 | 저평가 (매수 영역) |
| 중간 | 10-12배 | 적정 |
| 상단 | 12-14배 | 고평가 (차익실현 고려) |
| 초과 | 14배+ | 과열 |

## 3. 외국인 순매수 해석

| 패턴 | 의미 |
|------|------|
| 순매수 5일+ 연속 | 외국인 매수세 확인, 상승 추세 지지 |
| 순매수 10일+ 연속 | 강한 매수세, 추세 지속 가능성 높음 |
| 순매도 5일+ 연속 | 이탈 시작, 하락 압력 |
| 순매도 10일+ 연속 | 본격 이탈, 방어적 포지션 |
| 단일일 5,000억+ 순매수 | 이벤트 드리븐 (MSCI, 배당 등) |
| 단일일 5,000억+ 순매도 | 글로벌 리스크오프 가능성 |

## 4. 원달러 환율

| 레벨 (USD/KRW) | 상태 | 시장 영향 |
|:--------------:|------|-----------|
| <1,200 | 원 강세 | 내수주↑, 수출주↓, 외국인 매수↑ |
| 1,200-1,300 | 보통 | 균형 |
| 1,300-1,400 | 원 약세 | 수출주↑, 외국인 이탈 우려 |
| 1,400+ | 원 급약세 | 시장 불안, 정부 개입 가능 |

## 5. 한미 금리차 (10년)

| 차이 (한국-미국) | 의미 |
|:---------------:|------|
| +50bp 이상 | 한국 금리 높음, 원화 매력 |
| 0 ~ +50bp | 균형 |
| -50bp ~ 0 | 약간 역전, 외국인 채권 이탈 주의 |
| -50bp 이하 | 금리 역전, 자본 유출 압력 |

## 6. 신용잔고 / 투자자 예탁금

| 지표 | 고수준 의미 | 저수준 의미 |
|------|------------|------------|
| 신용잔고 | 레버리지 과열 (위험) | 시장 안도 |
| 투자자 예탁금 | 대기 자금 풍부 (상승 잠재력) | 자금 이탈 |
