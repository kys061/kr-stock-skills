# 한국 스킬 리뷰 루브릭

## 1. Metadata & Use Case (20%)

### 체크 항목
- SKILL.md 파일 존재 여부
- 사용 시점 섹션 존재
- 트리거 키워드 명시
- 관련 스킬 링크

### 채점
- 3/3 = 100점, 2/3 = 67점, 1/3 = 33점, 0/3 = 0점

## 2. Workflow Coverage (25%)

### 체크 항목
- 실행 단계 설명
- 입출력 정의 (함수 시그니처)
- 오류 처리 가이드

### 채점
- 3/3 = 100점

## 3. Execution Safety (25%)

### 체크 항목
- 실행 명령 예시 존재
- 절대/상대 경로 사용 일관성
- 재현 가능한 결과 보장

### 채점
- 3/3 = 100점

## 4. Supporting Artifacts (10%)

### 체크 항목
- scripts/ 디렉토리 존재
- references/ 디렉토리 존재
- 상수 정의 (Python 상수)

### 채점
- 3/3 = 100점

## 5. Test Health (20%)

### 체크 항목
- tests/ 디렉토리 존재
- 테스트 파일 존재
- 테스트 통과율 (pytest 실행)

### 채점
- 존재 + 100% 통과 = 100점
- 존재 + 부분 통과 = 50-99점
- 미존재 = 0점
