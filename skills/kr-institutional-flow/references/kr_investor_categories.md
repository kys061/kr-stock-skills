# 한국 투자자 12분류 가이드

## PyKRX 투자자 분류 체계

한국거래소(KRX)는 투자자를 12개 유형으로 분류하여 일별 매매동향을 공개한다.

### 12분류 → 3분류 매핑

```
외국인 (Foreign)
└─ 외국인 (등록 외국인)

기관 (Institutional)
├─ 금융투자 (증권사)
├─ 보험
├─ 투신 (자산운용)
├─ 사모
├─ 은행
└─ 연기금/기금

개인 (Retail)
└─ 개인

기타 (Other)
├─ 기타법인
├─ 기타외국인
├─ 기타금융
└─ 국가/지자체
```

### 각 투자자 유형 특성

| 유형 | 투자 성향 | 정보력 | 영향력 |
|------|----------|:------:|:------:|
| 외국인 | 장기/펀더멘털 | 높음 | 매우 높음 |
| 금융투자 | 단기/자기매매 | 높음 | 높음 |
| 보험 | 장기/안정 | 중간 | 중간 |
| 투신 | 중기/펀드 | 높음 | 높음 |
| 사모 | 중단기/적극 | 높음 | 중간 |
| 은행 | 장기/보수 | 중간 | 낮음 |
| 연기금 | 장기/가치 | 높음 | 매우 높음 |
| 개인 | 단기/모멘텀 | 낮음 | 높음(규모) |

### 수급 해석 원칙

1. **외국인 + 기관 동반 매수** → 가장 강한 상승 시그널
2. **외국인 매수 + 개인 매도** → 전형적 상승 패턴 (Retail-Counter)
3. **외국인 + 기관 동반 매도** → 강한 하락 시그널
4. **개인 매수 + 외국인/기관 매도** → 약세 지속 가능성

### US 13F 대비 장점

- **속도**: 분기 45일 지연 → 당일/D+1 확인
- **빈도**: 분기 1회 → 매일
- **정밀도**: 포지션 스냅샷 → 매수/매도 분리 계산
- **커버리지**: 대형주 중심 → 전 종목
