# kr-institutional-flow

한국 주식 시장의 투자자별 수급 동향을 추적하고 분석하는 스킬.

## 개요

PyKRX 기반 일별 투자자 매매동향 데이터를 활용하여 외국인/기관/개인의 수급 패턴을 분석한다.
US의 SEC 13F 분기 데이터와 달리, 한국은 일별 매매동향을 즉시 확인할 수 있어 더 정밀한 추세 전환 감지가 가능하다.

## 핵심 기능

1. **투자자별 수급 분석** — 12분류 투자자 데이터를 외국인/기관/개인 3분류로 축약하여 분석
2. **외국인 수급 심화 분석** — 지분율 추세, 전환점 감지
3. **축적/이탈 패턴 감지** — 외국인+기관 동반 매수 vs 개인 매도 패턴
4. **4팩터 종합 스코어링** — 외국인(40%) + 기관(30%) + 일관성(20%) + 거래량(10%)

## 스코어링

### 가중치
| 팩터 | 가중치 | 설명 |
|------|:------:|------|
| Foreign Flow | 40% | 외국인 수급 시그널 |
| Institutional Flow | 30% | 기관 합계 수급 시그널 |
| Flow Consistency | 20% | 수급 일관성 (20일 기준) |
| Volume Confirmation | 10% | 거래량 동반 확인 |

### 등급
| 등급 | 점수 | 의미 |
|------|:----:|------|
| Strong Accumulation | 85-100 | 강력 축적 — 외국인+기관 동반 매수 |
| Accumulation | 70-84 | 축적 중 — 모니터링 |
| Mild Positive | 55-69 | 약한 긍정 — 추가 확인 필요 |
| Neutral | 40-54 | 중립 — 수급 방향 불명 |
| Distribution | 0-39 | 이탈 중 — 주의 |

### 한국 특화: Retail-Counter 보너스
개인 매도 + (외국인+기관) 매수가 5일 이상 지속 시 +10점 보너스 (100점 상한).

## US 원본 대비 차이점

| 항목 | US (13F) | KR (PyKRX) |
|------|----------|-----------|
| 데이터 지연 | 분기 45일 | 당일/D+1 |
| 발표 빈도 | 분기 1회 | 매일 |
| 신호 등급 | 3등급 | 7등급 |
| 특화 로직 | 없음 | Retail-Counter 보너스 |
