# 한국 배당 가치 투자 방법론

## 1. 한국 배당 시장 특성

### 배당 패턴
- **연 1회 배당**: 대부분의 상장사가 12월 결산, 연 1회 배당
- **중간배당**: 일부 대기업 (삼성전자 등) 분기 배당 도입 중이나 소수
- **배당락일**: 12월 마지막 거래일 2영업일 전 (통상 12월 27-28일)
- **배당금 지급**: 주주총회 후 1개월 내 (통상 3-4월)

### 배당수익률 분포
- KOSPI 평균: ~1.8% (2020-2024)
- 고배당주 기준: ≥ 2.5% (미국 ≥ 3.5% 대비 낮음)
- 배당 귀족주 (10년 연속 증가): ~20종목 내외

### 배당세
- **15.4%** 원천징수 (소득세 14% + 지방세 1.4%)
- 금융소득 2,000만원 초과 시 종합과세 (최대 49.5%)
- 미국과 달리 적격/비적격 구분 없음

## 2. 밸류에이션 기준 (한국)

### PER 기준
| PER | 한국 시장 평가 |
|-----|:------------:|
| ≤ 8 | 극단적 저평가 |
| 8-12 | 저평가 |
| 12-15 | 시장 평균 |
| 15-20 | 약간 고평가 |
| > 20 | 고평가 |

### PBR 기준
| PBR | 한국 시장 평가 |
|-----|:------------:|
| ≤ 0.5 | 극단적 저평가 (자산주) |
| 0.5-0.8 | 저평가 |
| 0.8-1.0 | 적정 |
| 1.0-1.5 | 약간 고평가 |
| > 1.5 | 고평가 |

## 3. 배당 지속가능성 평가

### 배당성향 (Payout Ratio)
- **< 30%**: 매우 보수적, 배당 증가 여력 큼
- **30-50%**: 건전, 성장+배당 밸런스
- **50-70%**: 보통, 이익 감소 시 압박 가능
- **70-80%**: 주의, 이익 변동에 취약
- **> 80%**: 위험, 배당 삭감 가능성

### 부채비율 (D/E Ratio)
- **< 50%**: 매우 건전 (무차입 경영에 가까움)
- **50-100%**: 건전
- **100-150%**: 보통 (한국 기업 평균 수준)
- **> 150%**: 주의

## 4. 한국 배당 가치주 투자 전략

### 매수 타이밍
1. **3-4분기**: 연말 배당 기대감 상승 전 선매수
2. **배당락 후**: 배당락 급락 시 재매수 (장기 보유 전략)
3. **시장 급락 시**: 배당 안전마진 확대

### 포트폴리오 구성
- 단일 종목 비중: 5-10%
- 업종 분산: 최소 3개 업종
- KOSPI/KOSDAQ 비중: 80:20 (안정성)
