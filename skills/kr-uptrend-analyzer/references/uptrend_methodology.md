# 업트렌드 분석 방법론

## 1. 업트렌드 판정

### 개별 종목 업트렌드 기준
1. **종가 > 200일 SMA** (핵심 조건)
2. **200일 SMA 기울기 > 0** (추세 확인)
3. **종가 > 50일 SMA** (보조 확인, 가점 부여)

### 업종별 업트렌드 비율
```
업트렌드 비율 = (업트렌드 종목 수 / 전체 종목 수) × 100
```

## 2. 한국 업종 분류

### Cyclical (경기민감)
반도체, 자동차, 철강, 화학, 건설, 조선, 기계

### Defensive (방어)
통신, 유틸리티, 필수소비재, 제약/바이오

### Growth (성장)
2차전지, 인터넷/플랫폼, IT서비스

### Financial (금융)
은행, 보험, 증권

## 3. 5-컴포넌트 스코어링

### 3.1 Market Breadth (30%)
전체 시장 업트렌드 비율 기반 점수.

| 비율 | 기본 점수 |
|:----:|:---------:|
| ≥70% | 90 |
| ≥55% | 70 |
| ≥40% | 50 |
| ≥25% | 30 |
| <25% | 10 |

보정: 3일 변화량 상승 시 +10, 하락 시 -10

### 3.2 Sector Participation (25%)
업트렌드 비율 > 50%인 업종 수 기반.

| 참여 업종 비율 | 점수 |
|:-----------:|:----:|
| ≥80% | 90 |
| ≥60% | 70 |
| ≥40% | 50 |
| ≥20% | 30 |
| <20% | 10 |

보정: 업종간 표준편차 높으면(>20) -10

### 3.3 Sector Rotation (15%)
경기민감 vs 방어 업종 균형.

| 상태 | 점수 |
|------|:----:|
| Cyclical > Defensive + 20pp | 80 (성장국면) |
| Cyclical > Defensive | 70 |
| 비슷 (±10pp) | 50 |
| Defensive > Cyclical | 30 |
| Defensive > Cyclical + 20pp | 20 (방어국면) |

### 3.4 Momentum (20%)
EMA(3) 스무딩된 전체 업트렌드 비율의 변화 속도.

| 기울기 | 점수 |
|--------|:----:|
| 급상승 (>5pp/3d) | 90 |
| 상승 (>0) | 70 |
| 횡보 (≈0) | 50 |
| 하락 (<0) | 30 |
| 급하락 (<-5pp/3d) | 10 |

### 3.5 Historical Context (10%)
히스토리 내 현재 점수의 백분위.

## 4. 경고 시스템

### Late Cycle Warning (-5점)
원자재/에너지 업종 업트렌드 > 경기민감 평균 AND > 방어 평균
→ 후기 사이클 징후

### High Spread Warning (-3점)
업종간 최대-최소 스프레드 > 40pp
→ 시장 참여 불균형

### Divergence Warning (-3점)
동일 그룹 내 표준편차 > 20pp
→ 그룹 내 분화 (예: 반도체 상승, 건설 하락)

복수 경고 시 총 감점에서 +1 할인 적용 (최대 -10)
