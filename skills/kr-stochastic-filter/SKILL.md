# kr-stochastic-filter: Stochastic Slow 필터

> Stochastic Slow (18,10,10) 기반 모멘텀 확인 필터.
> %K >= %D 조건을 판정하여 스크리너 스킬의 추가 확인 지표로 활용.

## 사용 시점

- 스크리너 스킬이 종목 추천 시 추가 확인 필터로 호출
- 단독 실행: 특정 종목의 Stochastic Slow 상태 확인

## 방법론

### Stochastic Slow (18,10,10) 계산

| 단계 | 수식 |
|------|------|
| Fast %K | (Close - 18일Low) / (18일High - 18일Low) x 100 |
| Slow %K | Fast %K의 10일 SMA |
| Slow %D | Slow %K의 10일 SMA |

### 판정 기준

| 조건 | 결과 | 의미 |
|------|------|------|
| Slow %K > Slow %D | PASS | 상승 모멘텀 |
| Slow %K = Slow %D | PASS | 모멘텀 전환점 (동일도 통과) |
| Slow %K < Slow %D | FAIL | 하락 모멘텀 |

### 보조 판정 (참고용)

| 영역 | 범위 | 의미 |
|------|------|------|
| 과매수 | %K > 80 | 상승 과열 |
| 중립 | 20 <= %K <= 80 | 일반 |
| 과매도 | %K < 20 | 반등 가능 |
