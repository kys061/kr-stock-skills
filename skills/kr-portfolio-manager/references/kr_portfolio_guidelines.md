# 한국 포트폴리오 관리 가이드

## 1. 자산배분 원칙

### 1.1 코어-위성 전략
- **코어 (60-70%)**: 대형 우량주, 인덱스 ETF
- **위성 (30-40%)**: 성장주, 테마 ETF, 중소형주

### 1.2 시장별 배분
- **KOSPI**: 안정적 대형주 중심
- **KOSDAQ**: 성장성 중심 (변동성 높음)
- **ETF**: 분산투자 및 섹터 노출

## 2. 리스크 관리

### 2.1 포지션 사이징
- 단일 종목 최대 15% (집중 리스크 방지)
- 단일 섹터 최대 35% (섹터 리스크 방지)
- 현금 비중 최소 5% (유동성 확보)

### 2.2 리밸런싱 주기
- **정기**: 분기별 점검
- **이벤트 기반**: 10% 이상 편차 즉시 조정
- **세금 효율**: 연말 세금 최적화

## 3. 한국 시장 특성

### 3.1 거래 비용
- 증권거래세 0.23% (매도 시)
- 중개수수료 0.015% (매수/매도 각)
- 슬리피지 약 0.1%

### 3.2 배당
- 3월 결산법인 배당 집중 (12월 배당락)
- 분기배당 확대 추세
- 배당소득세 15.4% 원천징수

### 3.3 대주주 양도소득세
- 종목당 10억원 이상 보유 시 적용
- 세율 22% (지방세 포함)
- 보유기간 2년 미만 시 33%

## 4. 포트폴리오 액션

| 액션 | 조건 |
|------|------|
