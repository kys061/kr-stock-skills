## 1. 배경 및 목적

YouTube 채널 "주식장인"의 이란-전쟁 후 시장 전략 분석 영상(ESL-KEnXZXQ)과 우리 kr-stock-skills 46개 스킬 시스템을 비교한 결과, **85% 정렬** 확인. 나머지 15% 갭을 보완하여 실전 투자자 관점의 완성도를 높인다.

### 핵심 원칙 (주식장인 프레임워크)
- "금리 사이클이 최상위 대전제" — 금리 방향이 모든 투자 판단의 출발점
- TGA 잔고가 유동성의 숨은 변수
- 고객 예탁금이 한국 시장의 대기 자금 지표
- "금리 인하기 × 단기 급락 = 최고의 매수 기회" (공포 매수 패턴)

## 2. 보완 항목 (3개)

### Item 1: TGA (Treasury General Account) 추적 — `liquidity_tracker`

| 항목 | 내용 |
|------|------|
| 대상 스킬 | `us-monetary-regime` |
| 대상 파일 | `skills/us-monetary-regime/scripts/liquidity_tracker.py` |
| 현재 상태 | 4-컴포넌트 (Fed B/S, M2, DXY, RRP) |
| 보완 내용 | TGA 잔고를 5번째 컴포넌트로 추가 |
| 근거 | TGA 잔고 감소 → 시중 유동성 증가 (재정지출), TGA 증가 → 유동성 흡수 (국채 발행). 주식장인: "TGA가 유동성의 숨은 변수" |
| 가중치 변경 | 기존 4컴포넌트 재배분 → 5컴포넌트 (Fed B/S 0.25, M2 0.20, DXY 0.20, RRP 0.15, TGA 0.20) |
| 파라미터 | `tga_balance_b` (조 달러), `tga_change_1m_pct` (1개월 변화율 %) |
| 스코어링 | TGA 감소 → 유동성 증가 (점수 상승), TGA 증가 → 유동성 감소 (점수 하락) |
| 테스트 | 기존 테스트 호환성 + TGA 신규 테스트 ~15개 |

### Item 2: 고객 예탁금 추적 — `kr-credit-monitor`

| 항목 | 내용 |
|------|------|
| 대상 스킬 | `kr-credit-monitor` |
| 대상 파일 | `skills/kr-credit-monitor/scripts/credit_monitor.py` |
| 현재 상태 | 신용잔고 중심 4-컴포넌트 (잔고비율, 담보비율, 반대매매, 레버리지사이클) |
| 보완 내용 | 고객 예탁금(투자자예수금) 컴포넌트 추가 |
| 근거 | 고객 예탁금 = 증권사에 예치된 대기 자금. 증가 → 매수 여력 풍부, 감소 → 자금 이탈. 주식장인: "예탁금이 시장 바닥 판단의 핵심" |
| 구현 방식 | 기존 4-컴포넌트에 5번째 `investor_deposits` 컴포넌트 추가 또는 별도 함수 |
| 파라미터 | `deposit_balance_t` (조원), `deposit_change_1m_pct` (1개월 변화율 %) |
| 스코어링 | 예탁금 증가 → 대기 매수 자금 풍부 (긍정), 예탁금 감소 → 자금 이탈 (부정) |
| 테스트 | 기존 테스트 호환성 + 예탁금 신규 테스트 ~10개 |

### Item 3: PANIC_BUY 패턴 — `kr-strategy-synthesizer`

| 항목 | 내용 |
|------|------|
| 대상 스킬 | `kr-strategy-synthesizer` |
| 대상 파일 | `skills/kr-strategy-synthesizer/scripts/strategy_synthesizer.py` |
| 현재 상태 | 4패턴 (POLICY_PIVOT, UNSUSTAINABLE, CONTRARIAN, WAIT) |
| 보완 내용 | 5번째 패턴 `PANIC_BUY` 추가 |
| 근거 | "금리 인하기(easing) × 단기 급락(-10% 이상) = 최고의 매수 기회". 주식장인: "공포에 사라" 원칙의 정량화 |
| 트리거 조건 | US regime = easing (score >= 65) AND KOSPI 단기 급락 >= -10% (20일 내) |
| 확신도 보너스 | PANIC_BUY 감지 시 확신도 +15pts |
| 추가 조건 | VIX/VKOSPI 급등(>25), 외국인 순매도 3일 연속, 신용잔고 안정(반대매매 아닌 순수 외부 충격) |
| 테스트 | 기존 테스트 호환성 + PANIC_BUY 신규 테스트 ~10개 |
