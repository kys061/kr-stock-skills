# Plan: kr-stock-analysis 매수구간/매도타점 기능 추가

> Feature: buy-sell-target
> Created: 2026-03-08
> Status: Plan

## 1. 배경 및 목적

현재 kr-stock-analysis 스킬은 종합 점수(0-100)와 투자 등급(STRONG_BUY~STRONG_SELL)을 제공하지만,
**구체적인 매수 가격대와 매도 가격대**를 제시하지 않는다.

사용자는 "언제 사야 하는가", "얼마에 팔아야 하는가"에 대한 명확한 가이드를 원한다.

### 참고 이미지 (사용자 제공)
- **매수구간**: 1차, 2차 분할매수 타점 제시
- **매도타점**: 1차 차익실현, 2차 보유물량 홀딩 등 유동적 대응전략

## 2. 요구사항

### 2-1. 매수구간 (Buy Zone)

| 항목 | 설명 |
|------|------|
| 1차 매수가 | 현재가 기준 1차 분할매수 추천 가격 |
| 2차 매수가 | 추가 하락 시 2차 분할매수 추천 가격 |
| 손절가 | 이탈 시 손절 기준 가격 |
| 매수 근거 | 해당 가격대의 기술적/펀더멘털 근거 |

### 2-2. 매도타점 (Sell Target)

| 항목 | 설명 |
|------|------|
| 1차 목표가 | 1차 차익실현 가격 (보유 물량의 50% 매도) |
| 2차 목표가 | 2차 차익실현 가격 (잔여 물량 매도) |
| 트레일링 스탑 | 고점 대비 하락 % 기준 |
| 매도 근거 | 해당 가격대의 기술적/밸류에이션 근거 |

### 2-3. 표시 형식

리포트의 새 섹션 "매수/매도 전략"으로 추가:

```markdown
## 13. 매수/매도 전략

### 매수구간

| 구분 | 가격 | 현재가 대비 | 근거 | 비중 |
|------|------|:----------:|------|:----:|
| 1차 매수 | 280,000 | -5.1% | MA60 지지, 볼린저 하단 | 50% |
| 2차 매수 | 260,000 | -11.9% | MA120 지지, 52주 저점 근접 | 50% |
| 손절가 | 240,000 | -18.6% | MA120 이탈 시 추세 전환 |  |

### 매도타점

| 구분 | 가격 | 현재가 대비 | 근거 | 비중 |
|------|------|:----------:|------|:----:|
| 1차 목표 | 330,000 | +11.9% | 컨센서스 목표가 하단 | 50% |
| 2차 목표 | 360,000 | +22.0% | 컨센서스 목표가 상단 | 잔량 |
| 트레일링 스탑 | 고점 -10% | - | 고점 갱신 후 하락 반전 시 | 전량 |

### 리스크/리워드 비율
| 항목 | 값 |
|------|-----|
| 예상 수익 (1차 목표) | +11.9% |
| 예상 손실 (손절) | -18.6% |
| R/R Ratio | 0.64:1 |
| 판단 | ⚠️ R/R 비율 불리 — 2차 매수가 진입 시 개선 |
```

## 3. 매수/매도 가격 산출 로직

### 3-1. 매수구간 산출

| 우선순위 | 데이터 소스 | 산출 방법 |
|:--------:|-----------|----------|
| 1 | 기술적 지표 (MA/BB) | MA60, MA120, 볼린저 하단을 지지선으로 활용 |
| 2 | 52주 고/저 | 52주 저점 대비 비율 기반 지지대 |
| 3 | 컨센서스 목표가 | 목표가 하단을 하방 지지 참고 |
| 4 | 투자 등급 연동 | BUY 등급 이상만 매수구간 표시 |

#### 1차 매수가 산출 규칙
```python
candidates = [ma60, bollinger_lower, recent_support]
buy_1 = max(candidates)  # 가장 가까운 지지선
# 현재가 대비 -3% ~ -10% 범위 내 필터
```

#### 2차 매수가 산출 규칙
```python
candidates = [ma120, week52_low * 1.05, buy_1 * 0.93]
buy_2 = max(candidates)  # 더 깊은 지지선
# 1차 매수가 대비 -5% ~ -15% 범위 내 필터
```

#### 손절가 산출 규칙
```python
stop_loss = min(ma120 * 0.97, buy_2 * 0.95)
# MA120 이탈 또는 2차 매수가 -5%
```

### 3-2. 매도타점 산출

| 우선순위 | 데이터 소스 | 산출 방법 |
|:--------:|-----------|----------|
| 1 | 컨센서스 목표가 | 목표가 평균/상단 활용 |
| 2 | 기술적 저항선 | 볼린저 상단, 52주 고가 |
| 3 | 밸류에이션 적정가 | 업종 평균 PER × EPS |
| 4 | R/R Ratio 기반 | 손절가 대비 최소 2:1 목표 |

#### 1차 목표가 산출 규칙
```python
candidates = [target_mean, bollinger_upper, resistance_1]
sell_1 = min(candidates)  # 가장 가까운 저항선
# 현재가 대비 +5% ~ +20% 범위 내 필터
```

#### 2차 목표가 산출 규칙
```python
candidates = [target_high, week52_high, sell_1 * 1.10]
sell_2 = min(candidates)  # 더 높은 저항선
```

#### 트레일링 스탑
```python
trailing_stop_pct = 10  # 기본 10%
# Beta > 1.5인 경우 15%로 확대
# Beta < 0.8인 경우 7%로 축소
```

### 3-3. 투자 등급별 표시 규칙

| 등급 | 매수구간 | 매도타점 |
|------|:-------:|:-------:|
| STRONG_BUY | ✅ 적극 매수구간 표시 | ✅ 목표가 표시 |
| BUY | ✅ 매수구간 표시 | ✅ 목표가 표시 |
| HOLD | ⚠️ "추가 매수 비추, 보유 유지" | ✅ 매도타점 표시 (보유자용) |
| SELL | ❌ 매수구간 미표시 | ✅ 매도타점만 표시 (청산 기준) |
| STRONG_SELL | ❌ 매수구간 미표시 | ✅ "즉시 청산 권고" |

### 3-4. R/R Ratio (Risk/Reward)

```python
expected_profit = sell_1 - current_price  # 1차 목표 기준
expected_loss = current_price - stop_loss
rr_ratio = expected_profit / expected_loss

# 판단:
# R/R >= 3.0 → "매우 유리"
# R/R >= 2.0 → "유리"
# R/R >= 1.0 → "보통"
# R/R < 1.0 → "불리 — 진입 재고 필요"
```

## 9. 제약사항 및 주의

1. **투자 조언 면책**: 리포트에 "투자 판단의 참고용이며, 실제 투자는 본인 책임" 문구 포함
2. **데이터 정밀도**: WebSearch 폴백 시 MA/BB 정확값 미확인 → "추정 기반" 표기
3. **적자 기업**: OPM 적자, PE 음수 기업은 밸류에이션 기반 목표가 산출 불가 → 기술적 지표만 활용
4. **무커버리지**: 애널리스트 0명 종목은 컨센서스 목표가 미사용 → 기술적+52주 범위만 활용
